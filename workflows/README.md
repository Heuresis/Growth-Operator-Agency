# `workflows/` — Operational workflows

Multi-skill, multi-agent processes that run a creator business end-to-end. Skills are atomic; workflows are how the atoms compose into operations.

| Subdirectory | Holds |
|---|---|
| [`client-onboarding/`](client-onboarding/) | Day-1 / Week-1 / 30-day / 90-day onboarding sequences |
| [`divisions/`](divisions/) | Per-division pipelines — foundations, marketing, nurture, sales, launch, scale, partnerships |
| [`delivery/`](delivery/) | How work moves from intake to shipped artifact |
| [`handoffs/`](handoffs/) | Inter-agent and inter-department handoff protocols |
| [`automations/`](automations/) | Trigger-driven flows: lead-to-CRM, purchase-to-onboarding, referral-trigger |
| [`operations/`](operations/) | Internal-ops cadences, tool SOPs, dashboards, team leadership, mode-of-operations |
| [`execution-templates/`](execution-templates/) | Fill-in templates for kickoffs, post-mortems, retros |
