## Appendix A — Mobile Adaptations

- Shorten hero copy for mobile
- Enlarge CTAs for thumb-tap
- Collapse long sections into accordions
- Ensure page loads in <3 sec on 4G
- Sticky CTA footer

## Appendix B — Technical Requirements

- Page speed target: <3 sec desktop, <4 sec mobile
- Conversion tracking: pixel + server-side event
- Split-test setup: variant A (control) vs variant B
- UTMs captured + persisted through checkout
- Accessibility: alt text on all images, semantic headings, color contrast WCAG AA

## Appendix C — Performance Targets

| Metric | Target |
|---|---|
| Above-fold engagement (scroll past hero) | >= 70% |
| Time on page (visitors past hero) | >= 2 min |
| CTA click rate (of scrolled-past-hero) | >= 15-25% |
| CTA-to-checkout / application rate | >= 40-60% |
| Page conversion rate (cold traffic) | 1-5% |
| Page conversion rate (warm / retargeting) | 5-15% |

## Appendix D — A/B Test Roadmap

- Hero headline A vs B
- Mechanism framing A vs B
- Stack order A vs B
- Guarantee length / emphasis A vs B
- FAQ placement — above vs below final CTA
- Price reveal — anchor-first vs anchor-last
