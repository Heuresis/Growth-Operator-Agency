## Structure Overview

| Segment | Duration | Objective |
|---|---|---|
| Pre-show lobby | 5-10 min | Warm up audience, set expectations, capture intent |
| Intro | 5 min | Hook + credential + promise + agenda |
| Secret 1 | 10-12 min | Teach contrarian principle 1 |
| Secret 2 | 10-12 min | Teach contrarian principle 2 |
| Secret 3 | 10-12 min | Teach contrarian principle 3 |
| Pivot | 3-5 min | Bridge from teaching to offer |
| Pitch | 15-20 min | Offer reveal + stack + guarantee + price + scarcity |
| Q&A | 15-20 min | Live objection handling |
| Close | 2-3 min | Final CTA, deadline, logistics |

## Pre-Show Lobby (T-10 to T-0)

**Objective:** Warm up attendees. Capture late arrivals. Set "you're in the right place" tone.

**Script loop (5-10 min rotation):**
> Hey everyone, welcome in. I see [FILL — archetype] joining from [FILL — cities / roles]. Say hello in the chat. Tell me [FILL — low-stakes prompt that surfaces ICP pain: "what's the one thing you're hoping to solve today?"].
>
> We'll start at [time] sharp. While you wait, [FILL — pre-show action: download the workbook / follow the host / bookmark the replay link].

**Pre-show assets:**
- Workbook PDF link in chat
- Host social handle
- Replay-access expectation set

## Appendix E — Evergreen Adaptation Notes

For evergreen delivery:
- Pre-show lobby → pre-recorded animated countdown
- Live chat engagement beats → "type in the comments" (async capture)

- Scarcity → real cohort-based, not fabricated countdown
