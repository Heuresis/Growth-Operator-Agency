# Marketing — Shared Knowledge

## Purpose
Shared context for every content and acquisition skill in the Marketing division. Defines channel mix, content pillars, voice-match rules, cadence, and the specificity bar.

## Division Skills (7)
- `/ad-creative` — paid ad variants (Meta, YouTube, TikTok)
- `/content-calendar` — 4-week rolling calendar across channels
- `/write-reel` — short-form Reel/TikTok script (30–90s)
- `/write-youtube` — long-form YouTube script (8–25 min)
- `/write-linkedin-post` — LinkedIn text / carousel post
- `/write-x-thread` — X/Twitter thread (6–12 posts)
- `/story-sequence` — IG/FB story sequence (5–10 frames)

## Core Vocabulary
- **Core Four (acquisition-economics methodology)** — the four acquisition channels: (1) warm outreach, (2) post free content, (3) cold outreach, (4) run paid ads. Every client runs ≥ 2.
- **Pillar Split 40/30/20/10** — content allocation: 40% Authority (frameworks, teaching), 30% Narrative (stories, case studies), 20% Proof (results, screenshots), 10% CTA (offers, launches).
- **Specificity Score (1–10)** — how concrete the claim is. "Grew faster" = 2. "Grew MRR from $8.4K to $47K in 11 weeks" = 9. Gate: ≥ 8.
- **Hook-Retention-Payoff** — three-part structure every piece must satisfy: hook ≤ 3s or ≤ 1 line, retention loop across the middle, payoff in the last 20%.
- **Voice-Match Score (1–10)** — blind-test a sample against the Voice Guide. Gate: ≥ 8.
- **CTR / Hook Rate / 3s View** — platform-native leading indicators of hook quality.

## Channel Cadence (default)
| Channel | Cadence | Format Mix |
|---|---|---|
| YouTube long-form | 1/week | 1 pillar + 1 shorts spin-off |
| Reels / TikTok | 5–7/week | 3 authority + 2 narrative + 1 proof + 1 CTA |
| LinkedIn | 5/week | 3 text + 1 carousel + 1 video |
| X / Threads | 7–14/week | 1 thread + 3–5 replies + 1–2 quote-posts daily |
| IG Stories | Daily | 5–10 frame sequence |
| Email | 3/week minimum | Mon teach / Wed narrative / Fri CTA |
| Paid Ads | Always-on | 5 creatives tested / week minimum |

## Voice-Match Rules (inherits from `/extract-voice`)
1. Use the signature phrase bank (min 1 per piece for long-form, min 1 per 3 pieces for short-form).
2. Never use banned words (listed in `voice-guide.md`).
3. Match sentence-length distribution within ±15%.
4. Use the founder's rhetorical patterns (rule-of-3, contrast pairs, concrete→abstract).
5. Default POV is first person singular unless the pillar calls for teaching plural.

## Patterns We Use
- **Hook bank**: every skill pulls from `reference/swipe-file/hooks.md`. Minimum 5 hook options drafted per piece, 1 chosen.
- **One claim, one proof**: no claim ships without a specific number, date, or named subject.
- **The Zoom rule**: one piece = one idea. If you can't say it in one sentence, split it.
- **Repurpose chain**: every YouTube → 3 Reels + 2 LinkedIn + 1 X thread + 3 Story frames + 1 email.
- **Launch-week override**: during launches, shift to 10/20/20/50 split (CTA-heavy). Default pillar mix resumes post-launch.

## Anti-patterns (block these)
- Vague claims ("massive growth", "game-changer")
- Hook that doesn't map to payoff
- Voice-mismatch (generic LinkedIn-ese)
- Pillar-mix drift (3 weeks of authority, zero proof)
- Ads without a single unit of specificity (number, name, screenshot)
- Calendar with < 7-day lead time
