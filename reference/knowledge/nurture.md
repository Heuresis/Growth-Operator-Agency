# Nurture — Shared Knowledge

## Purpose
Shared context for every lifecycle / middle-of-funnel skill. Nurture converts captured attention into booked calls and retained customers via sequenced email, lead magnets, and webinars.

## Department Skills (4)
- `/email-sequence` — multi-touch email sequences (welcome, indoctrination, re-engagement, sales)
- `/lead-magnet` — tripwire / opt-in asset (PDF, swipe file, mini-course, checklist)
- `/webinar-script` — live / evergreen webinar (perfect-webinar pattern)
- `/post-booking-nurture` — drip between call-booked and call-held

## Core Vocabulary
- **Indoctrination** — the first 5–7 emails after opt-in: belief-shifts, not sells.
- **Show Rate** — booked calls that actually show. Target ≥ 70% (live), ≥ 55% (cold traffic webinars).
- **Close Rate** — booked-and-shown calls that convert. Target ≥ 25% for high-ticket.
- **Unsubscribe Rate** — target ≤ 0.5% per send.
- **Deliverability Score** — inbox placement %. Target ≥ 95% (primary inbox).
- **Objection Map** — ranked list of top 8 objections for the offer. Every objection has ≥ 1 email that dissolves it.
- **8 Required Beliefs** (inherits from Foundations) — sequences must touch all 8 by the sales email.
- **Perfect Webinar (Russell Brunson)** — Hook → Story → Content (3 Secrets) → Stack → Close. Default for webinar scripts.

## Cadence Rules
- **Welcome sequence**: 5 emails across 7 days. Day 0 (deliver), Day 1 (story), Day 2 (mechanism), Day 4 (proof), Day 6 (soft CTA).
- **Indoctrination**: daily for 7 days, then 3x/week for 3 weeks.
- **Re-engagement**: triggered after 30 days no-open. 3-email sequence, then suppress if no open.
- **Post-booking**: 6 touches between book and call (email + SMS). Day 0 confirm, Day 1 prep, Day 2 story, Day 3 proof, -24h reminder, -1h reminder.
- **Webinar runway**: 7-day registration → event. Touches on register day, +1, +3, event-day morning, 1h before, at-start.

## Compliance (non-negotiable)
- Every email has a physical mailing address (CAN-SPAM).
- Every email has a one-click unsubscribe link (CASL, GDPR, CAN-SPAM).
- No buying lists. Consent-based opt-in only.
- Confirmed (double) opt-in for EU/UK audiences.
- Affiliate disclosure in any email with an affiliate offer (FTC).
- Never sell/share email lists without opt-in re-confirmation.
- Healthcare / financial niches: add disclaimer block per local regulator.

## Objection-Handling Patterns
| Objection | Pattern | Example Frame |
|---|---|---|
| "Won't work for me" | Case study with closest-match ICP | "Here's a [same role/niche] who..." |
| "I don't have time" | Time-compression proof | "2 hours/week was enough because..." |
| "Too expensive" | ROI math + cost-of-inaction | "Every month you delay = $X lost" |
| "I've tried similar" | Mechanism contrast | "What makes this different is..." |
| "Not sure it's the right time" | Urgency without manipulation | "Here's what changes on [date]..." |
| "I can figure it out myself" | Time/risk honesty | "You can. It took me 3 years and $47K in mistakes." |
| "My situation is unique" | Pattern-break | "Three edge cases we've solved..." |
| "I need to think about it" | Decision framework | "Here's how I'd decide in your shoes..." |

## Patterns We Use
- **One email, one idea, one CTA** — no multi-CTA emails.
- **Subject-line specificity** — use a number, name, or curiosity gap every time.
- **Plain-text bias** — HTML templates only when visually required (course drip, launch).
- **Soft-then-hard CTA** — first 3 emails soft (reply / read), last 2 hard (book / buy).
- **Tag on behavior** — clicks, opens, replies all trigger segmentation.

## Anti-patterns (block these)
- Sending to unengaged segments (kills deliverability)
- 5+ links in a single email
- "Just checking in" — always provide a reason to reply
- Webinars > 90 minutes (retention collapses)
- Post-booking nurture with zero SMS (show rate drops 15%+)
