## How this directory relates to `reference/frameworks/youtube/`

The 15 files in `reference/frameworks/youtube/` are platform-general YouTube frameworks synthesizing the content OS director, the VSL copywriter, the acquisition economist, and creator-native methodology lineage. This directory is the backend economist-specific: the 3 C's, the 4 Competencies, the Scientific Method protocol, and the 8-module sequencing are the backend economist's blueprint structure. Where overlap exists (retention logic, thumbnail testing, hook structure), the the backend economist files cross-reference the platform-general files rather than restate them.

**Do not conflate:**
- `youtube/7-video-types.md` (the content OS director's flywheel — 7 video types by funnel stage) ≠ anything in this directory
- `youtube/3-tier-funnel.md` (the content OS director's Top / Middle / Bottom funnel) ≠ `3-cs-system.md` (the backend economist's Click / Consume / Convert retention funnel)
- `youtube/vssl-15-step.md` (the VSL copywriter's VSSL) ≠ the backend economist's long-form 3-act structure (which lives in `8-module-architecture.md` under Module 4)
