# Instagram Profile Funnel — The the paid media director Stack

> **Reference:** Siguard (Siggy) the paid media director — Stallion Media / Automated IG Ecosystem
> **Purpose:** IG as a direct-response funnel — profile-as-landing-page, ads stack, DM-setter flow, content studio.
> **When to use:** Marketing (IG-driven acquisition), Sales (profile-funnel mechanics), Ads (IG-specific creative).

## Core Thesis — "Profile = Landing Page"

Most creators treat their Instagram profile as a **branding showcase**. the paid media director treats it as a **landing page with no back button**.

Every element of the profile converts:
- Name + handle = domain
- Bio = headline + subhead
- Link = CTA
- Highlights = trust elements / social proof
- Grid = "body copy" of the landing page
- Stories = live engagement layer

## The Automated IG Ecosystem

Five integrated components:

### 1. Profile Funnel (top of ecosystem)
Profile optimized to convert from first-view. Structure:
- **Name field:** Your name | Your category/niche (searchable keywords)
- **Bio line 1:** Who you help + transformation (1 line)
- **Bio line 2:** How you're different (mechanism / positioning)
- **Bio line 3:** Social proof or authority marker
- **Bio line 4:** Explicit CTA with emoji-arrow
- **Link:** Lead magnet or offer
- **Highlights:** Logical journey (About / Testimonials / Results / Process / Offer / Free Training)

### 2. Content Studio (Stallion Media approach)
Weekly content production system:
- **Reels:** 3-5/week, 60-90s, hook in first 3s, value-dense
- **Stories:** 2-4 daily, rotating through 4-layer anatomy (text + image + proof-screenshots + visuals)
- **Carousels:** 2-3/week, educational, save-optimized (10-15 slides)
- **Main feed posts:** rare — let Reels carry volume; feed for pinned hero content

### 3. Detailed Ads Skills (core differentiator)
Not generic "run IG ads." Specific ad types with specific functions:

- **Profile Funnel Ad** — ad that sends traffic to PROFILE (not link), audience consumes profile, follows, later buys. Lowest-friction acquisition.
- **Retargeting Ads** — sequence after profile visit or website visit
- **Video Ad (Hook Style)** — first 3 seconds do the work; bold claim + scroll-stopper
- **Video Ad (Story Style)** — narrative arc, 60-90s, identity-shift framing
- **Video Ad (Testimonial)** — raw client video, no polish, specificity-heavy
- **Image Ad (Single)** — high-contrast thumbnail with text hook
- **Carousel Ad** — 3-5 slides of storytelling + CTA
- **UGC-Style Ad** — mimics organic content, reduces "I'm being sold to" resistance

### 4. DM Setter Flow (Infomatic Setting Flow)
Once a lead comes from the profile funnel or ad:
- Setter identifies intent marker (comment, DM, profile follow)
- Setter responds within 5 min with value-add (not pitch)
- Qualifying question (2-3 max)
- If qualified: booking link or calendar pop
- If not qualified: nurture tag, no more DMs

### 5. Client Portal V12 (post-sale infrastructure)
Post-sale delivery environment:
- Sprints DB (what they're working on)
- Recurring Tasks (their weekly commitments)
- Master Sprints (90-day arcs)
- Content System (templates they use)
- Asset Upload Zone (their work product)
- Appointment Setter Dashboard
- Payments DB

## The 4-Layer IG Story Anatomy (the stories director + the paid media director hybrid)

Every story has 4 layers working simultaneously:
1. **Text** (script engine) — the message/claim
2. **Image** (background/proof/connection) — NEVER blank — subconscious proof layer
3. **Proof-Screenshots** (trust) — Stripe screenshots, testimonials, DMs (proof, not claims)
4. **Visuals** (charts/graphs/Miro) — authority layer, shows process

Never blank backgrounds. Never claim without screenshot. Mix orange (heavy/persuasive) and grey (light/casual) — 7 heavy sequences = too salesy.

## The Ad → Profile → DM → Call Path

```
Meta Ad (Profile Funnel Ad, $0.16/profile view)
  ↓
Profile (landing-page-optimized)
  ↓
Free guide or Lead Magnet (from link or from DM)
  ↓
Setter DM (within 5 min)
  ↓
Qualifying question
  ↓
Booking link
  ↓
Call
  ↓
Close
```

Each stage has a KPI target:
- Ad → profile visit: $0.16-$0.80 target
- Profile visit → follow: 15%+
- Profile visit → DM/click: 5%+
- DM → qualified: 40%+
- Qualified → booked: 70%+
- Booked → showed: 60%+
- Showed → closed: 25%+ for $3-5K offers, 15%+ for $10-25K

## YouTube Planner Addition (Instagram profile-funnel methodology)

the paid media director also runs a YouTube planner integrated with IG — videos feed long-form to IG profile as "further" for serious prospects. IG does discovery, YouTube does nurture, profile is the hub.

## Anti-Patterns

- ❌ Running generic engagement ads (vs Profile Funnel Ads) — wastes spend on people who never see the offer
- ❌ Polished "branded" posts without specificity or proof — won't convert
- ❌ Hiring IG managers who think "branding" = no CTA — every post needs a conversion path
- ❌ Using IG without YouTube / email / LinkedIn to back up — IG alone caps at nurture depth
