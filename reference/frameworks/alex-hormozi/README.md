## Invariants across all 5 PDFs

1. Value is a ratio. Cut the denominator before padding the numerator.
2. There are four lead-gen channels. Stop inventing a fifth.
3. The maybes are where the money is. Train closing, don't avoid it.
4. Speed wins. 5 minutes to contact, 72 hours to appointment.
5. Customers generate better ads than founders.
6. Volume negates luck.
