# Pre-Script Brief Template (1-Page)

> **Why this matters:** 80% of YouTube video failures are briefing failures. The script is a symptom — the brief decides whether the video has a job to do. A creator who fills out this 1-page brief before scripting saves 3–5 hours per video and ships tighter content.

## Source / Lineage

- the content OS director (brief-first video production)
- the acquisition economist (pre-production briefs from acquisition.com)
- Standard YouTube agency pre-production workflow

## The Template

Every video — Short or long-form — starts with this brief, completed BEFORE any scripting, shooting, or editing begins.

```
================================================================
VIDEO BRIEF — [Date] — [Working Title]
================================================================

1. GOAL (pick one)
   [ ] Reach / top-of-funnel (Tier 1)
   [ ] Nurture / authority (Tier 2)
   [ ] Convert / book-a-call (Tier 3)

2. PILLAR (pick one)
   [ ] Education (40%)
   [ ] Authority (30%)
   [ ] Entertainment (20%)
   [ ] Offer (10%)

3. VIDEO TYPE (pick one)
   [ ] Educational Deep-Dive
   [ ] VSSL (5-chapter or 15-step)
   [ ] Lifestyle-Vlog
   [ ] Tutorial
   [ ] Case Study
   [ ] Listicle
   [ ] Reaction-Interview
   [ ] Short

4. ICP STAGE (who this is for)
   - Awareness level: [Unaware / Problem-aware / Solution-aware / Product-aware / Most aware]
   - Stage in funnel: [Cold / Warm subscriber / Hot lead]
   - Specific persona: [one-sentence description]

5. HOOK (write 3 options, pick 1)
   Option A: _______________________________________________
   Option B: _______________________________________________
   Option C: _______________________________________________
   Selected: [ A / B / C ]
   Hook formula used: [from hook-library-20-formulas.md]

6. PROMISE (one sentence)
   By the end of this video, the viewer will _______________.

7. MECHANISM (what they'll learn)
   - Name of framework/idea: ______________________________
   - Number of steps or components: _______________________
   - One-line summary: ____________________________________

8. PROOF (what you're showing as evidence)
   - [ ] Specific client case (name: ________, result: ________)
   - [ ] Your own numbers (source: ________)
   - [ ] Third-party data/endorsement (source: ________)
   - [ ] Live demo / worked example
   - [ ] Other: ___________________________________________

9. CTA (what you want the viewer to do next)
   Primary CTA: __________________________________________
   CTA type: [ Playlist / Lead magnet / Book-a-call / Newsletter / None ]
   Where: [ Mid-video / End-card / Description / All ]

10. LENGTH TARGET
    [ ] 0–60s (Shorts)
    [ ] 3–5 min (quick authority)
    [ ] 8–15 min (tutorial)
    [ ] 15–30 min (deep-dive)
    [ ] 30–45 min (VSSL 5-chapter)
    [ ] 45–90 min (VSSL 15-step / interview)

11. TITLE DIRECTION (draft 3, pick 1)
    Title A: ________________________________________________
    Title B: ________________________________________________
    Title C: ________________________________________________
    Selected: [ A / B / C ]
    Character count (max 60): ____

12. THUMBNAIL DIRECTION (from title-thumbnail-patterns.md)
    Pattern #: ____
    Visual elements:
    - Face emotion: _________________________________________
    - Object/screenshot: ____________________________________
    - Text overlay (3–5 words): _____________________________
    - Dominant color(s): ____________________________________

13. WHERE THIS FITS IN THE CHANNEL
    - Links TO this video from: _____________________________
    - Links OUT to next video: ______________________________
    - Playlist: _____________________________________________
    - Series (if applicable): _______________________________

14. SUCCESS METRICS (targets)
    - CTR target: _____ %
    - AVD target: _____ minutes
    - 50% retention target: _____ %
    - End-screen CTR target: _____ %
    - If Tier 3: book-a-call target: _____

15. NON-GOALS (what this video is NOT trying to do)
    _________________________________________________________
    _________________________________________________________

================================================================
Approved by: ___________________  Date: ________________
Script start date: _______________  Publish target: ________
================================================================
```

## How to Use the Brief

### Step 1: Fill in Section 1–4 first

These are strategic decisions. Get them wrong and everything below is wasted. If you can't pick ONE goal, ONE pillar, ONE type — the video isn't ready to brief. Come back when you can.

### Step 2: Hook before anything else

Section 5 — write 3 hooks before anything else. If none of the 3 hooks would make you click on YouTube browse, the video isn't worth scripting. Iterate until one hook passes the gut-test.

### Step 3: The Promise-Mechanism-Proof spine

Sections 6–8 are the video's spine. If you can articulate a clear promise, a named mechanism, and specific proof — the video will be strong regardless of editing. If any of those three is vague, fix it NOW.

### Step 4: Singular CTA

Section 9 must have ONE primary CTA. If you write two, pick one. If you write "all of the above," you've lost already.

### Step 5: Metrics as commitments

Section 14 isn't a wish list — it's a commitment. After the video is live for 30 days, review against these targets. Below target? Diagnose (see `5-phase-psychology.md` phase-failure diagnostic).

## Common Brief Failures

### "Vague Goal"
"Grow the channel." "Build authority." These aren't goals. A goal is: "Book 15 sales calls from this video in 60 days." Or: "Reach 500K new viewers and add 2K subscribers in 30 days."

### "Hybrid Type"
"It's a case study + deep-dive + VSSL." No. Pick one. Hybrids retain worse than committed single-type videos in almost every test.

### "No Mechanism"
The brief lists proof but no framework. Proof without mechanism = "here's what happened" without "here's why and how." Viewer learns nothing transferable. Fix: name the mechanism.

### "Two CTAs"
"Book a call AND download the PDF." The split destroys both. Pick one.

### "Length Mismatch"
Brief says 25-minute deep-dive, script ends up being 11 minutes. Either the brief was wrong or the scripter cut corners. Recalibrate: either the topic is a tutorial, not a deep-dive, or the scripter needs to add mechanism depth.

## Brief Review Cadence

- Solo operator: self-review before scripting. 10 minutes.
- With team: producer + operator review together. 20 minutes.
- Quarterly: pull all briefs from the last 12 videos, correlate to actual performance. Which brief elements predicted performance? Adjust templates.

---

## Integration with Scripts and Production

The brief feeds directly into:
- The script outline (promise, mechanism, proof become chapter headings)
- The shot list (hook scene, proof scene, CTA scene)
- The thumbnail design (emotion, object, text overlay already decided)
- The title (selected from 3 options)
- The description (first 2 lines come from promise, middle comes from mechanism summary)

A good brief means the script practically writes itself. A bad brief means the script will be stuck at draft 4 because the foundation is weak.
