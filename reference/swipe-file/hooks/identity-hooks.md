# Identity Hooks

> **20 identity-based hooks.** Speak to who the reader is — or who they want to become. Invitation, not pitch.

---

## How Identity Hooks Work

People read content that reflects or shapes their self-concept. "Who is this for?" is the unasked first question. Identity hooks answer it in line 1 so the wrong reader closes the tab and the right reader leans in.

Three identity patterns:
1. **The Mirror** — describe who they are with precision
2. **The Ladder** — describe who they're becoming
3. **The Tribe Call** — invoke "people like us do things like this"

---

## Variant Structure Template

```
[IDENTITY MARKER — age, role, stage, situation, or belief]
[DISTINGUISHING CONDITION — what separates this reader from the wrong reader]
[OPTIONAL: promise or invitation that only lands for the right identity]
```

Never hedge. If the hook tries to include everyone, it includes no one.

## The Hooks

### Pattern: The Mirror

**1. Age + Stage + Ambition**
```
If you're a [AGE]-year-old [ROLE] who [STAGE MARKER], this post is written directly for you.
Everyone else: scroll.
```
Platform fit: LinkedIn, email SL.

**2. Role + Constraint**
```
[ROLE TYPE] with [SPECIFIC CONSTRAINT — small list, no team, limited budget, one product]: read this.
Built for the shape you're in, not the shape gurus pretend you're in.
```
Platform fit: LinkedIn, email SL, YouTube.

**3. Current State Mirror**
```
You run [BUSINESS TYPE]. You're stuck at [REVENUE/METRIC PLATEAU]. You've tried [N THINGS].
This isn't the [N+1]th thing. It's a different category of move.
```
Platform fit: LinkedIn, YouTube, VSL.

**4. Situation Mirror**
```
[ICP] at [SPECIFIC INFLECTION POINT — first $10k, first hire, first launch plateau]: this is your moment to [SPECIFIC MOVE].
```
Platform fit: LinkedIn, YouTube, email SL.

**5. Belief Mirror**
```
If you already believe [FOUNDATIONAL BELIEF], you're in the right room.
If you don't, you'll want to scroll.
```
Platform fit: LinkedIn, YouTube, email SL.

**6. Rejection Mirror**
```
You've refused [COMMON BUT BAD PRACTICE] despite pressure.
This post is for the ones who refused — it validates the refusal and shows the alternative.
```
Platform fit: LinkedIn, X, email SL.

**7. The Minority Mirror**
```
You're in the [SMALL PERCENTAGE]% who [UNUSUAL BEHAVIOR OR CONDITION].
This entire post is calibrated for you, not the majority.
```
Platform fit: LinkedIn, email SL.

### Pattern: The Ladder

**8. Who You're Becoming**
```
The version of you at [FUTURE STATE] doesn't [CURRENT BEHAVIOR].
They do [FUTURE BEHAVIOR]. The shift starts now or it doesn't.
```
Platform fit: LinkedIn, email SL, VSL.

**9. Identity Graduation**
```
You're graduating from [OLD IDENTITY] to [NEW IDENTITY].
The old identity asks different questions. So does the old advice. Here's the new frame.
```
Platform fit: LinkedIn, YouTube, email SL.

**10. The 10× Self**
```
The [10×] version of you wouldn't [CURRENT ACTIVITY].
They'd [10× ACTIVITY]. The bridge is one decision this week.
```
Platform fit: LinkedIn, email SL, VSL.

**11. Next Level Who**
```
At the next level of [ICP BUSINESS], the operators don't [CURRENT BEHAVIOR].
They [NEXT-LEVEL BEHAVIOR]. Start behaving like the next level now — the numbers follow.
```
Platform fit: LinkedIn, YouTube, email SL.

**12. Permission To Evolve**
```
You've outgrown [PAST IDENTITY/STRATEGY]. You can tell because [SIGNAL].
Giving yourself permission to evolve is the move.
```
Platform fit: LinkedIn, email SL, IG stories.

### Pattern: The Tribe Call

**13. People Like Us**
```
[ICP TYPE] like us don't [COMMON BEHAVIOR CATEGORY DOES].
We [ALTERNATIVE BEHAVIOR]. Here's why it works for us and not them.
```
Platform fit: LinkedIn, YouTube, X.

**14. The Small Room**
```
There's a small room of [ICP TYPE] who [UNUSUAL PRACTICE].
I'm in it. You might be. Here's what happens in the room.
```
Platform fit: LinkedIn, email SL, YouTube.

**15. Fellow Operators**
```
To the [ICP TYPE] who [SHARED CONDITION — built without funding, hit $N without ads, stayed solo past $N]:
This one is for us.
```
Platform fit: LinkedIn, email SL.

**16. The Anti-Category**
```
We aren't [CATEGORY DEFAULT]. We never will be.
The strategy for us is different. Here's the map for the anti-category.
```
Platform fit: LinkedIn, YouTube, X.

**17. Quiet Builders**
```
If you build quietly — no tweetstorms, no hype cycles, no posing — this is your playbook.

```
Platform fit: LinkedIn, YouTube, email SL.

**18. Veterans + Operators**

Platform fit: LinkedIn, YouTube, email SL.

### Pattern: Identity Shift Marker

**19. The Moment Of Identification**
```
Read the next sentence. If it lands, you're [IDENTITY X]. If not, scroll.

```
Platform fit: LinkedIn, email SL, IG stories.

**20. The Identity Contract**
```
Signing up to be [NEW IDENTITY] means you accept [N SPECIFIC TERMS].
If any term is a dealbreaker, the identity isn't for you. Here are the terms.
```
Platform fit: LinkedIn, email SL, webinar open.

## Identity Specificity Rule

"Entrepreneur" is not an identity. "37-year-old consultant who just replaced a 6-figure salary" is an identity.

An identity marker must include at least two of:
- Age or age range
- Role or title
- Stage or milestone
- Situation or constraint
- Belief or refusal

Three+ = sharper. More readers filter out; the right readers lean all the way in.

---

## Voice-Match Priority

Identity hooks pull heavily from Compartment 2:
- `demographic_profile` → age, role, income, stage
- `psychographic_profile` → beliefs, fears, ambitions
- `jtbd_statement` → job the reader hires the offer to do

When the identity marker mirrors the creator's ICP doc verbatim, the hook lands with zero friction.

---

## Combination Rule

Identity hooks pair with:
- Problem-agitate hooks (identity + pain specific to that identity)
- Insight hooks (identity + reframe for that identity)
- Story hooks (identity + narrative that rhymes with identity)

Never pair with specific-result hooks in the hook slot — the identity call is already the filter; adding proof overfills.
