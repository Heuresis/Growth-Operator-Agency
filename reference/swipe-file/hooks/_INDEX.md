## When To Pick Which Category

### Use Curiosity when
- The topic is net-new to the reader
- You want saves + bookmarks
- Format is long-ish (LinkedIn 1300-2200 char, thread, carousel, YouTube 3-8 min)

### Use Contrarian when
- The reader has tried the category-accepted way and failed
- You want polarization + long comments
- The post will reference a Big Enemy (positioning doc)

### Use Problem-Agitate when
- The reader is in pain but hasn't named it
- The asset is in the nurture block (email sequence, VSL open, webinar open)
- Specificity will be the transition point from emotion to logic

### Use Specific Result when
- You have proof (numbers, dated screenshots, invoices)
- The asset is a case study or decision-stage post
- The reader is solution-aware or product-aware

### Use Identity when
- Speaking to a specific archetype (38-year-old founder, 4-day-week coach)
- Building community / cult methodology
- Inviting "people like us do things like this"

### Use Insight when
- You want to teach + reframe
- The pattern is counter-intuitive ("why less X = more Y")
- You're positioning for authority + saves

### Use Question when
- You want replies / engagement / DMs
- Format is IG story, Reel prompt, or short thread
- Conversion stage is awareness and you need signals

### Use Story when
- The body is narrative (adversity → catalyst → aftermath)
- Format supports 1300-2200 chars or 60-120 sec runtime
- You're humanizing the creator (personal → business bridge)

## Specificity Rule

Every hook in this swipe file is written for a Specificity Score ≥ 7 **when slots are filled**. The template scaffolds specificity; the filler adds the exact number, person, timeframe, outcome.

Example:
- Template: `I generated [$SPECIFIC_AMOUNT] in [$TIMEFRAME] using [$UNUSUAL_METHOD].`
- Filled: `I generated $47,200 in 11 days using a one-page PDF and a single DM script.`
- Unfilled on publish: **REJECT.**
