# Contrarian Hooks

> **25 contrarian hooks.** Position against a category-accepted belief. Create polarization = comments = reach.

---

## How Contrarian Hooks Work

LinkedIn Dec 2025 Knowledge Graph rewards long comments (10+ words). Twitter/X rewards quote-tweet debate. Reels reward watch-time spikes from pattern interrupt. All three are amplified by taking a stance against a default category belief.

Three contrarian patterns:
1. **Reversal** — the category says X, I say not-X
2. **Target Callout** — the ICP themselves do the wrong thing
3. **Frame Replacement** — not just "X is wrong," but "here's the better frame"

---

## Variant Structure Template

```
[CATEGORY-ACCEPTED BELIEF]
[YOUR REVERSAL]
[OPTIONAL: why your reversal holds — mechanism tease]
```

Keep the reversal sharp. Mild contrarian = no engagement.

## The Hooks

### Pattern: Reversal

**1. Pure Reversal**
```
[CATEGORY-ACCEPTED BELIEF] is wrong.
The opposite is true for [ICP]. Here's why.
```
Platform fit: LinkedIn, X, YouTube title.

**2. "Stop Doing X"**
```
Stop [ACTIVITY EVERYONE DOES].
[ICP] who [OPPOSITE ACTIVITY] outperform [N:1 ratio] over [TIMEFRAME].
```
Platform fit: LinkedIn, X, Reels.

**3. "Everyone Is Wrong About"**
```
Everyone is wrong about [TOPIC].
The evidence is obvious once you look at [SPECIFIC DATA / CASE].
```
Platform fit: LinkedIn, YouTube, X.

**4. The Inversion**
```
What [CATEGORY] calls [POSITIVE THING] is actually [NEGATIVE OUTCOME].
What they call [NEGATIVE THING] is the move.
```
Platform fit: LinkedIn, YouTube, X.

**5. Category Refusal**
```
I refuse to do [DEFAULT CATEGORY ACTIVITY].
Results since refusing: [SPECIFIC OUTCOME]. Here's what I do instead.
```
Platform fit: LinkedIn, X, email SL.

**6. Best Practice Myth**
```
The "best practice" in [CATEGORY] only worked when [PAST CONDITION].
[CURRENT CONDITION] has made it a trap.
```
Platform fit: LinkedIn, YouTube, X.

**7. Guru Refutation (archetype-only)**
```
The [ICP TYPE] telling you to [COMMON ADVICE] hasn't done it in [TIMEFRAME].
Here's what the operators currently doing it actually do.
```
Platform fit: LinkedIn, X, YouTube (careful — archetype only, no named individuals).

**8. Pattern Break**
```
The standard [PROCESS/SYSTEM] is optimized for [OUTDATED CONDITION].
Optimizing for [CURRENT CONDITION] requires doing almost every step in reverse.
```
Platform fit: LinkedIn, YouTube, X thread hook.

### Pattern: Target Callout

**9. Self-Sabotage Naming**
```
[ICP] sabotage themselves by doing [COMMON BEHAVIOR].
They call it [POSITIVE-SOUNDING NAME]. It's actually [NEGATIVE REFRAME].
```
Platform fit: LinkedIn, YouTube, X.

**10. Hard Mirror**
```
If you're [ICP] and you [DID SPECIFIC THING THIS WEEK], you're not the kind of [ICP] who hits [OUTCOME].
Here's the alternative.
```
Platform fit: LinkedIn, X, email SL.

**11. The Excuse You've Been Running**
```
[ICP]: you've been calling [BEHAVIOR] "[EXCUSE FRAME]."
It's actually [HONEST FRAME]. The difference is money.
```
Platform fit: LinkedIn, X, YouTube.

**12. Comfortable-Lie Callout**
```
The lie [ICP] tell themselves about [TOPIC]: "[LIE].
The truth: [OPPOSITE]. Below: what happens when you stop running the lie.
```
Platform fit: LinkedIn, X, email SL.

**13. Permission Withdrawal**
```
You don't get to call yourself [ASPIRED IDENTITY] if you [BEHAVIOR CONTRADICTING IT].
Either the behavior changes or the identity is performance.
```
Platform fit: LinkedIn, X, IG stories.

**14. The Activity That Feels Like Work**
```
[BEHAVIOR] feels like work.
It isn't. It's a dopamine loop shaped like work. Here's the actual work.
```
Platform fit: LinkedIn, YouTube, X.

**15. Effort ≠ Outcome Callout**
```
Most [ICP] confuse effort with outcome.
Effort is the ticket, not the prize. Below: the prize mechanism.
```
Platform fit: LinkedIn, X, YouTube.

## Polarization Calibration

Contrarian ≠ rude. Contrarian = **falsifiable disagreement with a category default.**

Safe polarization:
- Attack methods, not people
- Attack beliefs, not identities
- Attack structures, not individual operators
- Use archetypes ("the guru type," "the tactical operator") — never names

Unsafe polarization:
- Named individuals
- Gratuitous insults
- Claims without a mechanism
- Reversals that are just opposite for opposite's sake

---

## Combination Rule

Contrarian hooks pair well with:
- Insight hooks in the body (reversal → reframe → proof)
- Specific-result hooks in the body (reversal → dated evidence)

Never combine contrarian + contrarian. The second reversal weakens the first.
