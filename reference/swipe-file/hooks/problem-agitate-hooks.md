# Problem-Agitate Hooks

> **25 problem-agitating hooks.** Surface a pain the reader hasn't fully articulated. Emotional entry, logical exit.

---

## How Problem-Agitate Hooks Work

The reader has a vague discomfort. They've felt it for months. They haven't named it. A good problem-agitate hook names it first, then twists the knife with specificity.

Three agitation patterns:
1. **Naming The Ache** — articulate what the reader feels but hasn't said
2. **Cost Exposure** — show the hidden cost of the current state
3. **Future-Pace The Pain** — project forward to the cost if nothing changes

---

## Variant Structure Template

```
[NAME THE PAIN IN READER'S OWN LANGUAGE]
[AGITATE — cost, duration, or consequence]
[OPTIONAL: bridge to the hidden reason the pain exists]
```

Use `voice_of_customer` language. Use their exact complaint phrasing. Do not soften it.

## The Hooks

### Pattern: Naming The Ache

**1. The Unsaid Complaint**
```
The thing [ICP] don't say out loud: [SPECIFIC UNCOMFORTABLE TRUTH].
I said it. Now the real conversation.
```
Platform fit: LinkedIn, email SL, VSL open.

**2. The Mid-Year Honest Check**
```
You're [N] months into [GOAL] and you're still [UNWANTED STATE].
Be honest about why before reading the fix.
```
Platform fit: LinkedIn, email SL, VSL.

**3. The Replay Loop**
```
This is the [N]th time you've tried to fix [RECURRING PROBLEM].
Each attempt ended the same way. Here's what the pattern is telling you.
```
Platform fit: LinkedIn, email SL, VSL open.

**4. The Growing Gap**
```
The gap between where you said you'd be and where you are is bigger than it was last [TIMEFRAME].
Growing, not shrinking. The question is whether you're willing to see why.
```
Platform fit: LinkedIn, email SL, VSL.

**5. The Quiet Dread**
```
Monday morning. [ICP]. [STATE OF PAIN — e.g., staring at a content calendar with no hooks, a pipeline of zero calls, a bank balance that won't cover next month].
This hook is for the ones who know the exact feeling.
```
Platform fit: LinkedIn, email SL, webinar open.

**6. The Second-Rate Story**
```
You've been telling yourself you're [PERMISSIVE STORY].
You know the better story. The better story is hard. Here's how to move into it.
```
Platform fit: LinkedIn, email SL, VSL.

**7. Named Shame Point**
```
The part of your business you don't show publicly: [SPECIFIC WEAKNESS].
That's the part the fix starts with. Not the part you already show.
```
Platform fit: LinkedIn, email SL.

**8. The Phrase You Stopped Saying**
```
You used to say "[AMBITION STATEMENT]."
You stopped saying it. Not because it changed. Because you stopped believing it was possible. This is about getting it back.
```
Platform fit: LinkedIn, email SL, YouTube long-form.

**9. The Conversation You Avoid**
```
There's a conversation with [SPECIFIC PERSON — partner, team, client] you've avoided for [TIMEFRAME].
Not having it is costing you more than having it would.
```
Platform fit: LinkedIn, email SL.

### Pattern: Cost Exposure

**10. The Real Cost Of Free**
```
The "[FREE CONTENT/ADVICE]" you've been [CONSUMING] isn't free.
You paid with [TIME / CAPITAL / OPPORTUNITY]. Here's the ledger.
```
Platform fit: LinkedIn, email SL, X.

**11. The Opportunity Math**
```
Every [TIMEFRAME] you stay at [CURRENT STATE] costs $[SPECIFIC AMOUNT] in [SPECIFIC METRIC].
Compounding, not linear. Here's the compounding math.
```
Platform fit: LinkedIn, VSL, webinar open.

**12. The Burn Rate**
```
At current [BEHAVIOR RATE], you run out of [RESOURCE — cash, runway, energy, belief] by [SPECIFIC DATE].
If nothing changes. Something has to change.
```
Platform fit: LinkedIn, email SL, VSL.

**13. The Hidden Tax**
```
[COMMON ACTIVITY] has a hidden tax of [SPECIFIC COST].
You've been paying it for [TIMEFRAME]. Total paid: [SUM]. Below: how to stop paying.
```
Platform fit: LinkedIn, YouTube, X.

**14. The Silent Killer**
```
[BEHAVIOR] doesn't feel expensive.
It's silently consuming [RESOURCE] at [RATE]. Here's the audit.
```
Platform fit: LinkedIn, YouTube, email SL.

**15. The Slow Leak**
```
Your [SYSTEM] has a slow leak.
It's been leaking [AMOUNT] per [TIMEFRAME] for [DURATION]. Plugging it is the highest-ROI move this quarter.
```
Platform fit: LinkedIn, email SL, VSL.

**16. The Three-Year Cost**
```
If you keep [CURRENT BEHAVIOR] for 3 more years, the cumulative cost is [LARGE NUMBER].
Zoom out. Then choose.
```
Platform fit: LinkedIn, YouTube long-form, email SL.

### Pattern: Future-Pace The Pain

**17. Same Date Next Year**
```
Same date, one year from now.
If nothing changes, you're [EXTRAPOLATED STATE]. Close this tab or read on.
```
Platform fit: LinkedIn, email SL, VSL.

**18. The Fork**
```
Two paths from here.
Path A: [CURRENT TRAJECTORY EXTRAPOLATED]. Path B: [ALTERNATIVE TRAJECTORY]. The fork is this week.
```
Platform fit: LinkedIn, email SL, VSL.

**19. The Regret Timeline**
```
[N] years from now, the regret won't be "I tried and it didn't work."
It'll be "I never tried because [CURRENT EXCUSE]." Here's how to kill the excuse.
```
Platform fit: LinkedIn, YouTube, email SL.

**20. The 5-Year Version Of You**
```
The 5-year version of you is watching the decision you make this month.
They're either grateful or they're not. You decide now which.
```
Platform fit: LinkedIn, YouTube, email SL.

**21. Eulogy Projection**
```
Project forward to the last chapter of [AMBITION AREA].
The chapter nobody writes is "[SPECIFIC MISSED STEP]." Write it now or write it in regret.
```
Platform fit: LinkedIn, YouTube long-form.

**22. The Extrapolation**
```
At current pace:
— [METRIC 1] in [TIMEFRAME]: [NUMBER]
— [METRIC 2] in [TIMEFRAME]: [NUMBER]
— [METRIC 3] in [TIMEFRAME]: [NUMBER]
None of those numbers are the life you told people you wanted.
```
Platform fit: LinkedIn, carousel, VSL.

Platform fit: LinkedIn, email SL, VSL.

```
The thing you said to yourself in the shower this morning: "[UNCOMFORTABLE PHRASE].
You know the sentence. Here's what to do with it.
```
Platform fit: LinkedIn, email SL, VSL open.

## Agitation Ethics

Problem-agitate hooks work by surfacing real pain — the pain has to be real. Violations:
- Manufacturing pain the ICP doesn't feel = dishonest
- Agitating pain without providing a path = sadistic
- Using fear without mechanism = spam
- Naming pain + pivoting to an unrelated offer = bait-and-switch

Rule: every problem-agitate hook must bridge to a real mechanism in the body. If you agitate without mechanism, you built a trap.

---

## Voice-Match Priority

Problem-agitate hooks are the highest-leverage place to use `voice_of_customer` verbatim phrases. The reader reads their own words back and commits.

Pull directly from Compartment 2:
- `voice_of_customer.pain_language` → "pain I live in"
- `voice_of_customer.surface_complaints` → the complaint they've voiced
- `voice_of_customer.deep_fears` → the fear they haven't voiced

---

## Combination Rule

Problem-agitate pairs with:
- Specific-result hooks (pain → proof someone escaped it)
- Insight hooks (pain → hidden mechanism causing it)
- Identity hooks (pain → who you become by solving it)

Never pair with another problem-agitate. Double agitation = reader closes tab.
