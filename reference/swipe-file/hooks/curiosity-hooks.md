# Curiosity Hooks

> **30 curiosity hooks.** Open a knowledge gap the reader cannot stand leaving unclosed. Slot-based. Platform-tagged.

---

## How Curiosity Hooks Work

The reader's brain logs an open loop. The brain hates open loops more than it hates bad content. Close the loop in the body, earn the scroll-through.

Four curiosity patterns recur:
1. **The Unexpected Number** — a figure that breaks category expectation
2. **The Unresolved Story** — a scene without its ending
3. **The Missing Variable** — "X worked because of Y, not Z"
4. **The Hidden Mechanism** — "the real reason X happens"

Each template below maps to one of these patterns.

---

## Variant Structure Template

Every hook in this file follows the same scaffold:

```
[ATTENTION EARNER]            — number, named entity, or bold claim
[CURIOSITY TRIGGER]           — the gap
[OPTIONAL SECOND LINE]        — widens the gap before "see more" cutoff
```

Swap slots. Match length to platform. Keep the gap open.

## The Hooks

### Pattern: Unexpected Number

**1. Unexpected Magnitude**
```
I [DID THING] [SPECIFIC TIMES] in [TIMEFRAME].
Not one of the [TIMES] went the way I expected.
```
Platform fit: LinkedIn, X, YouTube short. Bridges to teaching body.

**2. Disproportionate Outcome**
```
[SMALL INPUT] turned into [LARGE OUTCOME].
The math only works because of one thing most [ICP] miss.
```
Platform fit: LinkedIn, X thread hook, email SL.

**3. Surprising Ratio**
```
[PERCENTAGE]% of [$ METRIC] came from [UNEXPECTED SOURCE].
[REMAINING PERCENTAGE]% came from everything else combined.
```
Platform fit: LinkedIn, X, carousel cover slide.

**4. Timeframe Compression**
```
[RESULT THAT USUALLY TAKES YEARS] in [UNUSUALLY SHORT TIMEFRAME].
Here's the part nobody tells you about how.
```
Platform fit: LinkedIn, YouTube short, email SL.

**5. Bankroll Reveal**
```
$[SPECIFIC AMOUNT] generated from [UNUSUAL ASSET].
Cost to produce: $[LOW NUMBER]. Time: [HOURS].
```
Platform fit: Reels, YouTube short, X.

**6. Revenue-Per-Hour Shock**
```
[HOURS WORKED] hours of actual work. [$ REVENUE] collected.
The hour-to-dollar ratio is what I want to explain.
```
Platform fit: LinkedIn, X, IG stories.

**7. Volume Without Credential**
```
[METRIC] [ICP] clients, [TIMEFRAME], and I still don't have [CREDENTIAL MOST ASSUME REQUIRED].
```
Platform fit: LinkedIn, YouTube long-form title.

### Pattern: Unresolved Story

**8. Interrupted Scene**
```
I was [DOING MUNDANE THING] when [UNEXPECTED INTERRUPTION].
What happened next changed how I [VERB + NOUN] forever.
```
Platform fit: LinkedIn, YouTube, email SL.

**9. Cliffhanger Open**
```
Three minutes before [SIGNIFICANT EVENT], I almost [ACTION THAT WOULD HAVE CHANGED OUTCOME].
```
Platform fit: LinkedIn, YouTube, email SL.

**10. Unfinished Decision**
```
The day I decided to [MAJOR DECISION], I had exactly [$ AMOUNT] in the bank.
I'll tell you what happened next, but the first decision is the part worth studying.
```
Platform fit: LinkedIn, YouTube, email SL.

**11. Moment-Before-The-Break**
```
Nine months of [GRIND ACTIVITY]. One conversation that broke the pattern.
This is the conversation.
```
Platform fit: LinkedIn, YouTube.

**12. Opening With The End**
```
By [SPECIFIC DATE], I had [OUTCOME OPPOSITE OF INTUITION].
Here's how I got there — starting from [OPPOSITE STARTING POINT].
```
Platform fit: LinkedIn, YouTube long-form.

**13. The Quiet Before**
```
Two weeks before [BIG RESULT], [UNEXPECTED INTERNAL STATE — e.g., almost quit, didn't want to open my laptop, etc.].
The thing that flipped it is what I want to explain.
```
Platform fit: LinkedIn, YouTube, email SL.

### Pattern: Missing Variable

**14. The Real Reason**
```
Everyone thinks [COMMON CAUSAL CLAIM] is why [OUTCOME].
It's not. It's [SURPRISING VARIABLE]. Here's the evidence.
```
Platform fit: LinkedIn, YouTube, X thread.

**15. Wrong Credit**
```
Credit for [OUTCOME] goes to [OBVIOUS FACTOR].
It shouldn't. [HIDDEN FACTOR] did [PERCENTAGE]% of the lift.
```
Platform fit: LinkedIn, X, YouTube.

**16. Variable Swap**
```
[ACTION A] doesn't produce [OUTCOME]. 
[ACTION B] does — and most [ICP] never try B because A is already the default.
```
Platform fit: LinkedIn, X, carousel.

**17. Control Group Nobody Ran**
```
I ran the same [SYSTEM] with and without [DEBATED COMPONENT].
The results removed the debate.
```
Platform fit: LinkedIn, YouTube, X.

**18. Correlation Break**
```
[COMMONLY LINKED THINGS A AND B] don't actually produce each other.
[HIDDEN THIRD FACTOR C] produces both. Here's the chain.
```
Platform fit: LinkedIn, YouTube.

**19. Reverse Causality**
```
[EFFECT] doesn't follow [ASSUMED CAUSE].
[ASSUMED CAUSE] follows [EFFECT]. And that reorders how to operate.
```
Platform fit: LinkedIn, X, YouTube.

**20. The Invisible Lever**
```
There are [N] levers on [SYSTEM]. Everyone sees [N-1]. 
The one nobody looks at is the one doing the work.
```
Platform fit: LinkedIn, YouTube, X.

### Pattern: Hidden Mechanism

**21. The Mechanism Behind**
```
[SURFACE-LEVEL RESULT] happens because of [INVISIBLE UNDERLYING MECHANISM].
Once you see the mechanism you stop debating the result.
```
Platform fit: LinkedIn, YouTube, X thread.

**22. Why This Specific Number**
```
The reason [METRIC] hit exactly [NUMBER] isn't luck or skill.
It's a [STRUCTURAL/MATHEMATICAL] constraint. Here's the math.
```
Platform fit: LinkedIn, YouTube, X.

**23. The Loop Nobody Maps**
```
[SYSTEM A] feeds [SYSTEM B] which feeds [SYSTEM C] which feeds [SYSTEM A].
That loop is why [OUTCOME]. Break any link, the whole thing stops.
```
Platform fit: LinkedIn, YouTube, carousel.

**24. Anti-Intuitive Mechanism**
```
The reason [ICP] don't get [OUTCOME] is the exact behavior they think gets them there.
Let me show you the cycle.
```
Platform fit: LinkedIn, YouTube, X.

**25. Load-Bearing Detail**
```
One [SMALL DETAIL] is doing [PERCENTAGE]% of the work in [SYSTEM].
Remove it — results evaporate. Most [ICP] don't know it exists.
```
Platform fit: LinkedIn, X, YouTube title.

**26. The Mechanism In One Sentence**
```
[OUTCOME] = [INPUT A] × [INPUT B] ÷ [HIDDEN FRICTION C].
If that equation is right, here's what it means for [ICP].
```
Platform fit: LinkedIn, carousel, X.

**27. Why Most Fail — Structurally**
```
[N]% of [ICP] fail at [GOAL] not because of effort.
The structure they inherit makes failure the default. Here's the structural fix.
```
Platform fit: LinkedIn, YouTube long-form.

**28. The Hidden Prerequisite**
```
Everyone teaches [SKILL X] as the path to [OUTCOME].
[SKILL X] only works if you already have [OVERLOOKED PREREQUISITE Y]. Here's Y.
```
Platform fit: LinkedIn, YouTube, X.

**29. The Tipping Threshold**
```
Below [THRESHOLD NUMBER], [SYSTEM] does not work.
Above it, [SYSTEM] compounds. The threshold is the game.
```
Platform fit: LinkedIn, YouTube, X.

**30. The Switch Point**
```
There's a single [EVENT/DECISION/METRIC] that flips [ICP] from [STATE A] to [STATE B].
Most spend years in State A without knowing the switch exists.
```
Platform fit: LinkedIn, YouTube, X thread.

## Voice-Match Notes

When filling these templates:
- Replace bracketed slots with the creator's own phrasing where possible
- Use `voice_of_customer` phrases from Compartment 2 when the reader-facing word is generic
- Check `phrases_to_avoid` list before publishing
- Specificity ≥ 8 on filled output — if the slot sits vague, the hook fails

---

## Combination Rule

Curiosity hooks combine well with:
- Problem-agitate hooks in line 2 (gap → pain)
- Specific-result hooks in line 2 (gap → proof)
- Insight hooks as the body that closes the loop

Do not combine curiosity hooks with other curiosity hooks in the same post. Compounding gaps exhaust readers.
