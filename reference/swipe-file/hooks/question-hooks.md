# Question Hooks

> **20 question-based hooks.** Force participation. Earn DMs, replies, and thumb-stop. Best for IG stories, Reels, short threads, email SL.

---

## How Question Hooks Work

A well-framed question does three things simultaneously:
1. Makes the reader pause (scroll-stop)
2. Prompts internal answer (participation)
3. Signals "this post is about something you care about" (filter)

Three question patterns:
1. **The Honest Audit** — uncomfortable truth-check question
2. **The Diagnostic** — question whose answer reveals where the reader is
3. **The Provocation** — question designed to generate replies/DMs

---

## Variant Structure Template

```
[QUESTION — 5-15 words, specific, uncomfortable or illuminating]
[SETUP LINE — frame the question or widen the stake]
[OPTIONAL: the "why I'm asking" or "where this goes"]
```

Avoid rhetorical fluff questions ("Ever wondered...?"). Readers see those instantly and tune out.

## The Hooks

### Pattern: The Honest Audit

**1. Self-Report Metric**
```
What's your [METRIC] right now?
Not the number you tell people. The real one. That number is the honest starting line.
```
Platform fit: LinkedIn, IG stories, email SL.

**2. Last-Time Question**
```
When was the last time you [IMPORTANT ACTIVITY]?
If the answer is "last week" — skip this post. If it's "[LONGER AGO]" — read on.
```
Platform fit: LinkedIn, IG stories, email SL.

**3. Compounding Audit**
```
How many [UNITS OF PROGRESS] have you produced in the last [TIMEFRAME]?
Multiply by [FUTURE FRAME]. That's your trajectory if nothing changes.
```
Platform fit: LinkedIn, email SL.

**4. The Honest-Hour Count**
```
How many hours a week do you actually spend on [HIGHEST-LEVERAGE ACTIVITY]?
Not [BUSYWORK LOOKALIKE]. The real thing. Be honest.
```
Platform fit: LinkedIn, IG stories, email SL.

**5. The Mirror Question**
```
If you had to describe your last [TIMEFRAME] in one word to a stranger — what's the word?
That word is a diagnostic. Here's what it tells us.
```
Platform fit: LinkedIn, IG stories, email SL.

**6. The Paid/Unpaid Audit**
```
In the last [TIMEFRAME], what percentage of your work was [REVENUE-GENERATING]?
The answer is usually lower than expected. Here's how to raise it.
```
Platform fit: LinkedIn, email SL.

### Pattern: The Diagnostic

**7. The Stage Diagnostic**
```
Which is your constraint right now — [STAGE CONSTRAINT A], [STAGE CONSTRAINT B], or [STAGE CONSTRAINT C]?
The answer determines every next decision.
```
Platform fit: LinkedIn, IG stories (poll), email SL.

**8. Fork Diagnostic**
```
Reading your [RESULT]: is it [INTERPRETATION A] or [INTERPRETATION B]?
Because the move is different for each.
```
Platform fit: LinkedIn, email SL.

**9. Scoring Question**
```
On a 1-10, how [QUALITY] is your [ASSET]?
If you're under [THRESHOLD], the next fix isn't [COMMON FIX]. It's [ACTUAL FIX].
```
Platform fit: LinkedIn, IG stories (slider), email SL.

**10. Which-Type Question**
```
Which [ICP TYPE] are you — [TYPE A], [TYPE B], or [TYPE C]?
Each type has a different highest-leverage move. Below: the map.
```
Platform fit: LinkedIn, IG stories (poll), YouTube.

**11. The Unlock Diagnostic**
```
What would have to be true for you to [BIG OUTCOME] in the next [TIMEFRAME]?
Work backwards from the answer. That's your plan.
```
Platform fit: LinkedIn, email SL.

**12. The Bottleneck Question**
```
What's the one constraint that, if removed, would [UNLOCK OUTCOME]?
Most [ICP] can't answer quickly. That inability is the first constraint.
```
Platform fit: LinkedIn, email SL.

### Pattern: The Provocation

**13. The Uncomfortable Direct Ask**
```
Be honest — if you stopped [COMMON BEHAVIOR] tomorrow, would [OUTCOME] actually get worse?
Most [ICP] have never asked. The answer is usually no.
```
Platform fit: LinkedIn, X, email SL.

**14. The Permission Question**
```
Who gave you permission to [UNDERCHARGE / UNDER-SELL / UNDER-CLAIM]?
Probably nobody. You gave it to yourself. Take it back.
```
Platform fit: LinkedIn, email SL.

**15. The Stop-Doing Ask**
```
What would you stop doing today if you trusted [PRINCIPLE] fully?
Write it down. That's the list.
```
Platform fit: LinkedIn, IG stories, email SL.

**16. The Counter-Permission**
```
What would happen if you [COUNTER-INTUITIVE ACTION] for [TIMEFRAME]?
The worst case is usually much less bad than the imagined worst case.
```
Platform fit: LinkedIn, email SL.

**17. The Identity Challenge**
```
If you truly were [ASPIRED IDENTITY], would you still [CURRENT BEHAVIOR]?
If no, the identity isn't real yet. It's performance.
```
Platform fit: LinkedIn, email SL.

### Pattern: The Hook-Plus-Participation

**18. Reply Trigger**
```
Quick question for [ICP]: is [BELIEF A] or [BELIEF B] closer to your experience?
Reply with A or B. I'll share what I'm seeing in the DMs.
```
Platform fit: LinkedIn, IG stories, X.

**19. Comment Trigger**
```
Comment [WORD X] if [CONDITION].
I'll DM the [RESOURCE] to anyone who fits.
```
Platform fit: LinkedIn, IG stories.

**20. The Poll Hook**
```
Poll: how many [ICP UNITS] are you currently [DOING THING]?
— [OPTION 1]
— [OPTION 2]
— [OPTION 3]
— [OPTION 4]
I'll break down what each range means.
```
Platform fit: LinkedIn (poll), IG stories (poll), X (poll).

## Question-Hook Avoidance List

Banned question patterns (they read as AI or low-effort):
- "Have you ever wondered why..."
- "What if I told you..."
- "Want to know the secret to..."
- "Did you know that..."
- "Can you believe..."

Safe question patterns:
- Direct self-audit
- Stage / type / bottleneck diagnostic
- Permission / identity challenge
- Explicit participation prompt (reply, comment, poll)

---

## Reply-Rate Calibration

Questions that drive replies share 3 traits:
1. **Binary or small option set** — easy to answer in 1 word
2. **Low stakes to answer** — no judgment on wrong answer
3. **Clear next step** — the asker promises to reply / send something

Questions that flop:
- Open-ended ("what do you think?")
- High-stakes ("what's your biggest fear?" — readers won't share publicly)
- No follow-through promise

---

## Combination Rule

Question hooks pair with:
- Problem-agitate hooks (question exposes pain)
- Identity hooks (question filters for identity)
- Insight hooks (question leads to insight reveal in body)

Never pair with another question. Compound questions dilute both.
