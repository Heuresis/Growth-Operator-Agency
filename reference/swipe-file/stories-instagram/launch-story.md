# 7-Day Launch Story Sequence — Instagram

> **Pre-cart → open → close.** 7-day story sequence for a specific offer launch. Warms, frames, and closes.

---

## When To Use

- Cart opens in 7 days
- Audience already knows the creator (warm)
- Offer is anchored to a specific open/close window
- Goal: maximize opt-ins + applications + sales

---

## The 7-Day Arc

| Day | Theme | Frames | Energy |
|---|---|---|---|
| Day -6 | Thesis tease | 5-6 | Curiosity |
| Day -5 | Problem articulation | 6-7 | Agitate |
| Day -4 | Case study / proof | 7-8 | Prove |
| Day -3 | Mechanism reveal | 6-7 | Teach |
| Day -2 | Offer preview + objections | 7-8 | Convert |
| Day -1 | Cart-open tomorrow | 5-6 | Urgency |
| Day 0 | Cart open day | 8-10 | Sell |

---

## Day -6 — Thesis Tease

**Hook (frames 1-2):**
```
Frame 1: [Bold text] "Something's coming. It took me [TIMEFRAME] to build."
Frame 2: [Face to camera, 5s] "Not ready to tell you everything yet. But I want to tell you why it exists."
```

**Value (frames 3-5):**
```
Frame 3: "Here's the problem I've been quietly solving for [ICP]..."
Frame 4: "I saw [SPECIFIC PATTERN] happen to [N]+ [ICP] this year."
Frame 5: "It happens because of [HIDDEN MECHANISM]. Most don't see it."
```

**Connection (frame 6):**
```
Poll: "Has this happened to you?" — Yes / No
```

**CTA (frame 7):**
```
"Tap the link to join the [PRE-LAUNCH LIST]. The details land there first."
[Link sticker]
```

## Days +1, +2 (follow-up until cart close)

Each day of the open window:
- Morning: reminder + testimonial
- Afternoon: objection-handling story
- Evening: capacity update + urgency

Do not spam — 1 story sequence per day max during open cart.

## Visual Consistency

Across the 7 days:
- Same color palette (2-3 colors max)
- Same font family
- Same text alignment convention
- Consistent use of creator's brand assets

---

## Post-Launch Stories

Day +3 (after cart closes):
- Thank you to buyers
- Honest note to non-buyers about what's next
- Soft transition to ongoing content cadence
