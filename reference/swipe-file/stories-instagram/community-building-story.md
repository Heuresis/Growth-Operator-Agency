# 7-Day Community Building Story Sequence — Instagram

> **Question-led. Participation-heavy.** 7-day sequence for weeks focused on audience signal + DM activation.

---

## When To Use

- Engagement rate plateau
- New followers need activation
- Collecting signal before building an offer
- Weeks without hard CTAs

---

## The 7-Day Arc

| Day | Theme | Participation Mechanism |
|---|---|---|
| Monday | Opening question of the week | Question box |
| Tuesday | Poll on a common [ICP] debate | Poll |
| Wednesday | Feature reader responses from Monday | Republish + comment |
| Thursday | Slider on a self-audit metric | Slider |
| Friday | Quiz — common misconception | Quiz |
| Saturday | Open DM invite around a specific topic | DM prompt |
| Sunday | Weekly recap of what the community said | Synthesis |

---

## Monday — Opening Question

**Hook:**
```
Frame 1: "Monday question."
Frame 2: "Real answers only. I'll feature responses Wednesday."
```

**Value:**
```
Frame 3: "The question: '[SPECIFIC OPEN-ENDED PROMPT — e.g., 'What's the hardest part of your business right now?']'"
Frame 4: "Why I'm asking: [HONEST REASON — e.g., 'I want to understand what's landing for you vs what isn't']"
Frame 5: "What counts as a good answer: [SPECIFIC — 'one honest sentence beats a paragraph of theory']"
```

**Connection:**
```
Frame 6: [Question box]: "[THE QUESTION]"
```

**CTA:**
```
"Wednesday: I'll feature the sharpest responses. Include your handle if you want credit."
```

---

## Tuesday — Poll

**Hook:**
```
Frame 1: "Quick poll."
Frame 2: "Debate in [ICP] circles right now. Curious where you land."
```

**Value:**
```
Frame 3: "Background: [BRIEF CONTEXT OF THE DEBATE]"
Frame 4: "Side A argues: [COMPRESSED A]"
Frame 5: "Side B argues: [COMPRESSED B]"
Frame 6: "Both have evidence. Curious what your data says."
```

**Connection:**
```
Frame 7: [Poll]: "Where do you land?" — Side A / Side B
```

**CTA:**
```
"I'll share the result distribution Thursday + what the minority answer teaches."
```

---

## Wednesday — Feature Responses

**Hook:**
```
Frame 1: "Monday's question. Sharpest responses here."
Frame 2: "Re-reading these clarified something for me. Sharing with credit."
```

**Value:**
```
Frame 3: "[Paraphrased response 1 with handle if given]"
Frame 4: "[My reaction — 1-2 sentences]"
Frame 5: "[Paraphrased response 2 with handle]"
Frame 6: "[My reaction]"
Frame 7: "[Paraphrased response 3 with handle]"
Frame 8: "[My reaction]"
Frame 9: "Pattern across responses: [SPECIFIC PATTERN I SAW]"
```

**Connection:**
```
Question box: "Did any of these hit? Which one?"
```

**CTA:**
```
"Tomorrow: slider question. Self-audit metric."
```

---

## Thursday — Slider

**Hook:**
```
Frame 1: "Slider question."
Frame 2: "Honest self-audit — no judgment."
```

**Value:**
```
Frame 3: "Context: [BRIEF WHY THIS METRIC MATTERS]"
Frame 4: "The metric: [SPECIFIC SELF-ASSESSMENT QUESTION — e.g., 'How clear is your positioning right now?']"
Frame 5: "What 1 looks like: [ANCHOR POINT]"
Frame 6: "What 10 looks like: [ANCHOR POINT]"
```

**Connection:**
```
Frame 7: [Slider]: "Where are you?" — 1-10
```

**CTA:**
```
"I'll share distribution Friday + what to do if you're below [THRESHOLD]."
```

---

## Friday — Quiz

**Hook:**
```
Frame 1: "Quiz."
Frame 2: "Common misconception among [ICP]. Which one's right?"
```

**Value:**
```
Frame 3: "Most [ICP] believe [COMMON CLAIM]."
Frame 4: "Is it true or false?"
```

**Connection:**
```
Frame 5: [Quiz]: "True / False"
Frame 6: "Correct answer: [ANSWER]"
Frame 7: "Here's why: [BRIEF EXPLANATION]"
Frame 8: "Evidence: [BRIEF PROOF]"
```

**CTA:**
```
"Save this if you've been operating on the wrong answer. Most [ICP] do."
```

---

## Saturday — Open DM Invite

**Hook:**
```
Frame 1: "Saturday. DMs open on one specific topic."
Frame 2: "[SPECIFIC TOPIC — narrow, not 'anything']"
```

**Value:**
```
Frame 3: "Why I'm opening DMs on this: [HONEST REASON]"
Frame 4: "What to send: [SPECIFIC PROMPT — e.g., '1 sentence about your current [PAIN], 1 sentence about what you've tried']"
Frame 5: "What I'll do: reply within 24h with 1-2 specific thoughts"
Frame 6: "Not a pitch. Just conversations."
```

**Connection:**
```
Frame 7: [DM prompt]: "Send 1 sentence."
```

**CTA:**
```
"I read every DM. Real conversations happen here."
```

---

## Sunday — Weekly Recap

**Hook:**
```
Frame 1: "Week in what the community said."
Frame 2: "Synthesizing what I heard across polls, DMs, responses."
```

**Value:**
```
Frame 3: "Top theme: [SPECIFIC PATTERN]"
Frame 4: "Surprise: [UNEXPECTED SIGNAL]"
Frame 5: "Where minority opinions clustered: [SPECIFIC]"
Frame 6: "What this tells me for next week's content: [PREVIEW]"
Frame 7: "Next week's opening question: [TEASE]"
```

**Connection:**
```
Question box: "What did I miss? Reply if a pattern stood out to you."
```

**CTA:**
```
"See you Monday. Real week — not the highlights one."
```

---

## Slot-Fill Rules

- Questions must be specific (not "how's everyone doing?")
- Featured responses require handle + permission if naming
- Paraphrase if not getting explicit permission
- Responses must be real (fabrication = violation)
- Slider anchors must be concrete

---

## Signal Collection Value

This sequence is worth running quarterly even without a hard conversion goal. Outputs:
- Current pain points in the audience (for content direction)
- Common misconceptions (for future contrarian posts)
- DM conversations (for offer refinement)
- Minority opinions (for category creation moves)

Document the week's signal in a simple doc — it feeds `company.yaml` updates to Compartment 2.

---

## Common Failure Modes

- Questions too vague ("how's your business?") — get vague answers
- Featured responses without permission / credit
- Fake paraphrases
- Saturday DM invite too broad (open-ended = low response rate)
- Sunday recap generic (not actually synthesizing)
