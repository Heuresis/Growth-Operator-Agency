# 7-Day Lead Magnet Story Sequence — Instagram

> **Drive DMs → email list → sales sequence.** 7-day story arc to promote one lead magnet.

---

## When To Use

- New lead magnet ready (PDF, checklist, mini-course, template)
- Goal: email list growth or DM funnel activation
- Soft top-of-funnel (not hard sell)
- Weekly cadence slot

---

## The 7-Day Arc

| Day | Theme | Purpose |
|---|---|---|
| Day 1 | Pain surfacing | Name the problem the magnet solves |
| Day 2 | Cost of the pain | Why this pain is expensive to ignore |
| Day 3 | The common mistake | What most [ICP] try and why it fails |
| Day 4 | Mechanism preview | Tease the lead magnet's core insight |
| Day 5 | Magnet drop | First DM trigger |
| Day 6 | Social proof | Who already grabbed it + results |
| Day 7 | Last chance (soft) | Final DM trigger + transition |

---

## Day 1 — Pain Surfacing

**Hook (frames 1-2):**
```
Frame 1: "[Specific pain in reader's own language — pulled from voice_of_customer]"
Frame 2: "If this hits, the rest of this week is for you."
```

**Value (frames 3-5):**
```
Frame 3: "Why this pain persists for most [ICP]..."
Frame 4: "It's not lack of effort. It's [STRUCTURAL REASON]."
Frame 5: "I wrote down the fix. Sharing this week."
```

**Connection (frame 6):**
```
Poll: "Does this pain describe where you are right now?" — Yes / Kind of / No
```

**CTA (frame 7):**
```
"Tomorrow: the real cost of leaving this unfixed. Tap notifications to see it."
```

## Day 2 — Cost of the Pain

**Hook:**
```
Frame 1: "Yesterday: the pain. Today: what it's costing you."
Frame 2: "This is the math most [ICP] never run."
```

**Value:**
```
Frame 3: "Per week, [PAIN] costs ~$[AMOUNT] in [METRIC]."
Frame 4: "Per year: $[AMOUNT × 52 rough]"
Frame 5: "Per 5-year horizon: $[CUMULATIVE SIGNIFICANT NUMBER]"
Frame 6: "Not linear. Compounding. Every unfixed week raises the baseline cost."
```

**Connection:**
```
Slider: "Where are you on the 'I've been carrying this pain' scale?" — 1 to 10
```

**CTA:**
```
"Tomorrow: the common fix that doesn't work (and why). Tap to follow."
```

## Day 3 — The Common Mistake

**Hook:**
```
Frame 1: "Most [ICP] try to fix [PAIN] with [COMMON SOLUTION]."
Frame 2: "It's the wrong solution. Here's why."
```

**Value:**
```
Frame 3: "[COMMON SOLUTION] treats the symptom."
Frame 4: "The actual cause is upstream: [REAL CAUSE]."
Frame 5: "Fixing the symptom without the cause = 6 months of motion, same result."
Frame 6: "The real fix starts at [SPECIFIC UPSTREAM POINT]."
```

**Connection:**
```
Question box: "Have you tried [COMMON SOLUTION]? What happened?"
```

**CTA:**
```
"Tomorrow: I'll tease the actual mechanism. Notifications on."
```


**Value:**

**CTA:**
```
"Tomorrow: I'm sharing the full worksheet. Free. Via DM. Tap to be ready."
```

## Day 5 — Magnet Drop (Primary DM Trigger Day)

**Hook:**
```
Frame 1: "Today's the day."

```

**Value:**
```
Frame 3: "What's inside — [SPECIFIC LIST OF 3-5 ITEMS]"

Frame 5: "Use it to [SPECIFIC OUTCOME + TIMEFRAME]"

```

**Connection:**
```
Frame 7: "Poll: Are you going to DM for it?" — Yes / Going to think about it
```

**CTA:**
```
Frame 8: "DM '[TRIGGER WORD]' now. I'll send it."
[Link sticker optional — "Or tap here to download directly"]
```

```

```

**CTA:**
```
"DM '[TRIGGER WORD]' if you haven't grabbed it yet. Link in bio too."
```

## Day 7 — Soft Final CTA + Transition

**Hook:**
```
Frame 1: "Last day I'll mention the [LEAD MAGNET] for a while."
Frame 2: "If you've been meaning to grab it, now's the moment."
```

**Value:**
```
Frame 3: "Why it's time-sensitive: [HONEST REASON — e.g., 'I'm moving to new content next week' or 'The context this week set up is fresh']"
```

**Connection:**
```

```

**CTA:**
```
Frame 6: "Final DM trigger: '[TRIGGER WORD]'. Or link in bio."
Frame 7: "Thanks for the week — real conversations happening in DMs. Come say hi."
```

## Slot-Fill Rules

- [LEAD MAGNET NAME] should be specific (not "my playbook" — "The 11-Question Client Intake Worksheet")
- [TRIGGER WORD] should be 1-2 syllables, easy to type, unique enough to track
- Social proof must be real (paraphrased OK; fabricated = violation)
- Specificity ≥ 8 on outcomes

---

## DM Funnel After The Trigger

When someone DMs the trigger word:
1. Auto-reply or hand-reply with the resource (within 1 hour)
2. Follow-up question 24h later: "Did you see the [SPECIFIC PART]? Most people miss it."
3. If reply comes: qualifying question for possible offer
4. If no reply: nurture email sequence kicks in (per `/email-sequence`)

See `write-linkedin-post` SKILL.md for the full comment-to-call funnel pattern applied to DM.

---

## Common Failure Modes

- Trigger word too generic (e.g., "yes" — won't track)
- Magnet description vague (readers don't know what they're DMing for)
- No social proof day (misses compounding)
- Stacking CTAs in frame 5-8
- Running two magnet sequences back-to-back (audience fatigue)
