# 7-Day Offer Tease Story Sequence — Instagram

> **Warming toward a higher-ticket consultative offer.** 7-day soft-pitch sequence. Runs 3-6 weeks before cart.

---

## When To Use

- Higher-ticket consultative offer in the works (weeks out from launch)
- Audience needs warming before direct pitch
- Creator wants to test signal before committing to launch
- Consultative ICP won't respond to hard-sell sequences

---

## The 7-Day Arc

| Day | Theme | Energy |
|---|---|---|
| Monday | Name the problem specifically | Agitate + identify |
| Tuesday | Show the structural reason it persists | Teach |
| Wednesday | Offer a free micro-solution (credibility) | Give |
| Thursday | Case arc — someone who solved it | Prove |
| Friday | Name your position relative to the problem | Position |
| Saturday | Open "first conversations" invite | Soft inquiry |
| Sunday | Quiet reflect — no pitch | Reset |

---

## Monday — Name the Problem

**Hook:**
```
Frame 1: "Naming something I hear from [ICP] every week."
Frame 2: "Not sure how many of you have this exact pain. Want to check."
```

**Value:**
```
Frame 3: "The pain: [SPECIFIC PAIN DESCRIBED IN READER'S OWN LANGUAGE]"
Frame 4: "Specific version: [NARROWER SLICE]"
Frame 5: "What it feels like day-to-day: [SENSORY / EMOTIONAL DETAIL]"
Frame 6: "What makes it worse: [COMPOUNDING FACTOR]"
```

**Connection:**
```
Poll: "Is this your specific pain right now?" — Exactly / Close / Not at all
```

**CTA:**
```
"Tomorrow: why this persists. The structural reason."
```

## Tuesday — The Structural Reason

**Hook:**
```
Frame 1: "Why [MONDAY'S PAIN] doesn't get solved."
Frame 2: "It's not effort. It's structure."
```

**Value:**
```
Frame 3: "The common attempts: [3 COMMON THINGS ICP TRIES]"
Frame 4: "Why each one plateaus: [1-LINE STRUCTURAL REASON PER ATTEMPT]"
Frame 5: "The real constraint: [UPSTREAM STRUCTURAL ISSUE]"
Frame 6: "Fixing the real constraint changes the math on everything downstream."
```

**Connection:**
```
Question box: "Which of the common attempts have you tried?"
```

**CTA:**
```
"Tomorrow: a free micro-fix. Works for ~40% of cases."
```

---

## Wednesday — Free Micro-Solution

**Hook:**
```
Frame 1: "Free micro-fix. No DM trigger. No email. Just apply it."
Frame 2: "Works for about 40% of cases. Full solution is harder."
```

**Value:**
```
Frame 3: "The move: [SPECIFIC SMALL ACTION]"
Frame 4: "How to run it: [STEPS]"
Frame 5: "Expected timeframe to see signal: [DURATION]"
Frame 6: "If it works: great, you're in the easier 40%"
Frame 7: "If it doesn't: you're in the 60% where the real constraint is upstream. That's a different conversation."
```

**Connection:**
```
Question box: "Going to try it? What would stop you?"
```

**CTA:**
```
"Tomorrow: someone who was in the 60%. How they solved it."
```

---

## Thursday — Case Arc

**Hook:**
```
Frame 1: "A [ICP] who was in the 60% — where the micro-fix didn't work."
Frame 2: "Their 90-day arc. Quick version."
```

**Value:**
```
Frame 3: "Starting state: [ANONYMIZED]"
Frame 4: "What they'd already tried: [SPECIFIC LIST]"
Frame 5: "The structural unlock: [UPSTREAM FIX WE WORKED ON]"
Frame 6: "90-day numbers: [SPECIFIC DELTA]"
Frame 7: "Why the micro-fix alone wouldn't have gotten here: [STRUCTURAL REASON]"
```

**Connection:**
```
Slider: "How close is this to your situation?" — Different / Similar / Exactly
```

**CTA:**
```
"Tomorrow: I'll name my specific position on this — what I do vs don't."
```

## Friday — Your Position

**Hook:**
```
Frame 1: "Naming what I actually do and don't do."
Frame 2: "Because the generic version of this is everywhere. Mine isn't generic."
```

**Value:**
```
Frame 3: "What I do: [SPECIFIC POSITIONING STATEMENT — 1 sentence]"
Frame 4: "Who I do it for: [SPECIFIC ICP]"
Frame 5: "What I don't do: [EXPLICIT ANTI-SCOPE]"
Frame 6: "Who I don't do it for: [ANTI-ICP]"
Frame 7: "Why the anti-scope matters: [BRIEF MECHANISM]"
```

**Connection:**
```
Poll: "Are you in the fit or out?" — In / Out / Not sure
```

**CTA:**
```
"Tomorrow: first-conversations invite. Small number, specific fit."
```

## Saturday — First Conversations Invite

**Hook:**
```
Frame 1: "Saturday. Soft invite."
Frame 2: "I'm opening [NUMBER] first conversations this month. Specific fit only."
```

**Value:**
```
Frame 3: "What a first conversation is: [BRIEF — e.g., '25-minute diagnostic call, no pitch, no followup unless we both want it']"
Frame 4: "What it's not: [ANTI-SCOPE — 'not a sales call, not a free coaching session']"
Frame 5: "Who it's for: [SPECIFIC FIT CRITERIA]"
Frame 6: "Who it's not for: [ANTI-ICP]"
Frame 7: "How to request: [SPECIFIC STEP — 'DM 'first' with 1 sentence about your situation']"
Frame 8: "I'll close requests at [SPECIFIC NUMBER] — usually fills in 24-48h."
```

**Connection:**
```
Frame 9: [DM prompt]: "DM 'first' — 1 sentence about where you are."
```

**CTA:**
```
"Link in bio has the context if you want the long version before DMing."
[Link sticker]
```

## Sunday — Quiet Reflect

**Hook:**
```
Frame 1: "Sunday. No pitch today."
Frame 2: "Quiet notes on what this week surfaced."
```

**Value:**
```
Frame 3: "What I heard in DMs: [GENERAL PATTERN, anonymous]"
Frame 4: "What I noticed in the polls: [PATTERN]"
Frame 5: "The surprising thing: [UNEXPECTED]"
Frame 6: "Next week: [PREVIEW OF NEXT THEME — not offer-related]"

```

**Connection:**
```
(none — Sunday is a deliberate reset)
```

**CTA:**
```
(none — leave space for the audience to breathe)
```

---

## Slot-Fill Rules

- Offer context must be real (don't tease an offer that isn't actually ready)
- Anti-scope must be genuine (not just a false-filter)
- First conversations must actually be set up (calendar ready, capacity set)
- No high-pressure urgency manufacturing

---

## The 60-40 Rule

In an offer tease sequence:
- 60% value / teaching / proof
- 40% offer-adjacent

Going above 40% offer-adjacent triggers audience fatigue. Going below 40% doesn't move conversion.

---

## First Conversation → Offer Bridge

When someone DMs "first":
1. Reply within 6 hours with a short qualifying question + calendar link (if qualified)
2. Run the 25-minute diagnostic as scheduled (no pitch)
3. At the end: "Based on what you shared, either it's a fit for [OFFER] or it isn't. Want me to walk you through it?"
4. If yes: outline offer; if no: close cleanly and send a specific resource

---

## Common Failure Modes

- Offer not actually ready (tease becomes vaporware)
- No anti-ICP specified (reduces credibility)
- Saturday invite too broad (not enough filter)
- Sunday reflect becomes another pitch
- Running this sequence while the audience is cold (requires existing warmth)
