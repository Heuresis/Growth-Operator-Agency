# 4-Layer Story Anatomy — Instagram

> **The frame every IG story sequence runs through.** Hook → Value → Connection → CTA. Per `/story-sequence` skill.

---

## The Four Layers

### Layer 1 — HOOK
**Purpose:** pattern interrupt. Earn the tap past the first frame.

**Frame count:** 1-2 frames (first 15-30 seconds of the sequence).

**Formats that work:**
- Bold text on bold background (high-contrast)
- Face to camera, first 2 seconds with expression + one sentence
- Visual before/after (split screen)
- Question to the camera (voice or text)
- Counter-intuitive claim in large type

**Formats that fail:**
- Logo / brand frame (readers tap past)
- Generic "hey guys" open
- Low-contrast text overlay
- Slow build (first 1-2 seconds must land)

---

### Layer 2 — VALUE
**Purpose:** deliver the payload. Teach, show, prove, entertain.

**Frame count:** 2-5 frames (the middle of the sequence).

**Formats that work:**
- Text slides with one idea per slide (3-5 slide mini-carousel)
- Talking head explaining a specific framework or moment
- Screen share / tutorial with clear steps
- Story / narrative with sensory detail
- Demo / before-after with numbers

**Pacing:**
- Each frame = one idea
- Visual rhythm break every 2 frames (change format)
- Specificity ≥ 8 on any claim

---

### Layer 3 — CONNECTION
**Purpose:** force participation. Turn viewers into respondents.

**Frame count:** 1-2 frames (near the end of the sequence).

**Formats that work:**
- Poll (2 options)
- Slider (1-10 on a specific question)
- Question box (open-ended but specific prompt)
- Quiz (one correct answer, playful)
- Reply prompt ("reply with a number from 1-5")

**Rules:**
- One connection prompt per day max
- Prompt must be specific (not "how are you doing?" — "which of these describes your week?")
- Creator replies to every response in DMs within 24h

---

### Layer 4 — CTA
**Purpose:** direct a single next action.

**Frame count:** 1 frame (the last or second-to-last).

**Formats that work:**
- Link sticker (direct tap → landing page)
- DM trigger ("send 'word' to my DMs")
- Swipe up to related story highlight
- "Tap to see the full post on the feed"

**Rules:**
- One CTA per day
- Matches the sequence type (launch sequence → cart link; lead magnet → DM trigger)
- Never stack ("follow + DM + click link + save")

---

## The Full Daily Frame Stack

```
Day N (5-8 frames total):

Frame 1 — HOOK (pattern interrupt)
Frame 2 — HOOK extension (widens tension)
Frame 3 — VALUE (payload 1)
Frame 4 — VALUE (payload 2)
Frame 5 — VALUE (payload 3)
Frame 6 — CONNECTION (poll / question box)
Frame 7 — CTA (single action)
Frame 8 — (optional) transition to tomorrow's sequence
```

---

## Frame-Level Specifications

### Visual
- 1080 × 1920 vertical
- Safe zones: top 220px (avatar), bottom 250px (reply bar)
- Text readable on 6" phone screen (minimum 24px)
- Consistent palette across the week
- Consistent font across the week

### Audio
- 70% of IG stories are watched without sound
- Always include text captions when using voice
- Music choice: matches brand energy (curated, not trendy)

### Duration
- Static frames: 5-7 seconds
- Video frames: 15 seconds max (the platform cap for a single segment)
- Total sequence: 2-4 minutes of runtime

---

## Metrics To Watch

| Metric | Signal | Healthy Range |
|---|---|---|
| Taps forward | boredom | <10% = strong |
| Taps back | interest (re-read) | >5% = strong hook |
| Exits | lost attention | <15% per frame |
| Replies | connection | 1-5% of views |
| Link taps | CTA strength | 2-8% of last-frame views |
| DM triggers | DM CTA strength | 3-10% of prompted views |

---

## Voice-Match Rules

- 3+ `phrases_to_use` across the week
- Banned vocabulary swept
- Specificity ≥ 8 on any claim
- Visual consistency (palette + font)

---

## Common Failure Modes

- No hook (first frame is a logo or generic greeting)
- Value buried (layer 2 starts at frame 5+)
- No connection prompt (stories without participation flatten)
- Multiple CTAs per day
- Inconsistent visual identity across the week
