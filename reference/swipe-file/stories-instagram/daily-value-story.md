# 7-Day Daily Value Story Sequence — Instagram

> **Pure authority build. No hard CTA.** 7-day sequence for weeks without specific promotions. Compounds presence.

---

## When To Use

- Nothing specific to sell this week
- Maintaining cadence between launches
- Audience authority compound
- New account warming (weeks 1-12)

---

## The 7-Day Theme Split

| Day | Theme | Pillar |
|---|---|---|
| Monday | Frame of the week | Insight |
| Tuesday | Teaching / How-To | Education |
| Wednesday | Behind the decision | Transparency |
| Thursday | Reader question / DM feature | Community |
| Friday | Proof / small case | Authority |
| Saturday | Lightweight / personal | Humanization |
| Sunday | Reflect + preview | Priming |

---

## Monday — Frame of the Week

**Hook:**
```
Frame 1: "The frame I'm running this week."
Frame 2: "One sentence. I'll expand it for the next 5 frames."
```

**Value:**
```
Frame 3: "[COMPRESSED INSIGHT — one sentence that could tattoo]"
Frame 4: "Why it matters for [ICP]: [1-line]"
Frame 5: "What it changes in practice: [1-line action]"
Frame 6: "Where I saw it break for someone recently: [brief anonymized example]"
Frame 7: "I'll be running everything this week through this frame."
```

**Connection:**
```
Poll: "Does this frame land?" — Yes / Pushing back
```

**CTA:**
```
"Save this to story highlights — come back mid-week."
```

---

## Tuesday — Teaching / How-To

**Hook:**
```
Frame 1: "How to [SPECIFIC SMALL OUTCOME] in [SHORT TIMEFRAME]."
Frame 2: "3 steps. Tactical. Executable today."
```

**Value:**
```
Frame 3: "Step 1: [Specific action + why]"
Frame 4: "Step 2: [Specific action + why]"
Frame 5: "Step 3: [Specific action + why]"
Frame 6: "Common mistake: [Specific trap]"
Frame 7: "Expected result: [Specific outcome, timeframe-bounded]"
```

**Connection:**
```
Question box: "Which step feels hardest for you?"
```

**CTA:**
```
"Save this. Come back when you're ready to run it."
```

---

## Wednesday — Behind the Decision

**Hook:**
```
Frame 1: "A decision I made this week that I'm uncertain about."
Frame 2: "Showing the logic so you can push back if you see the hole."
```

**Value:**
```
Frame 3: "The decision: [SPECIFIC DECISION]"
Frame 4: "The options I considered: [A, B, C]"
Frame 5: "Why I chose [CHOSEN OPTION]: [1-2 sentences]"
Frame 6: "What I might be wrong about: [honest acknowledgment]"
Frame 7: "I'll know in [TIMEFRAME] whether this was right."
```

**Connection:**
```
Question box: "What would you have done?"
```

**CTA:**
```
"Reply with your take — the pushback is welcome."
```

---

## Thursday — Reader Question / DM Feature

**Hook:**
```
Frame 1: "Question from the DMs this week:"

```

**Value:**
```
Frame 3: "My short answer: [1-sentence]"
Frame 4: "The longer version: [expansion]"
Frame 5: "Where most [ICP] go wrong on this: [common mistake]"
Frame 6: "What to actually do first: [specific action]"

```

**Connection:**
```
Question box: "What's your question? I'll pick one for next Thursday."
```

**CTA:**
```
"DMs are open. I read them."
```

## Friday — Proof / Small Case

**Hook:**
```
Frame 1: "A [ICP] I worked with this month:"
Frame 2: "Came in at [STARTING STATE]. Right now: [CURRENT STATE]."
```

**Value:**
```
Frame 3: "Starting numbers: [SPECIFIC METRICS]"
Frame 4: "First move we made: [SPECIFIC ACTION]"
Frame 5: "What surprised both of us: [UNEXPECTED OBSERVATION]"
Frame 6: "Current numbers: [SPECIFIC METRICS, dated]"
Frame 7: "Still in progress — not a finished case study. Early signal."
```

**Connection:**
```
Slider: "Where are you on the same path?" — 1-10
```

**CTA:**
```
"Follow for more honest mid-stream case notes."
```

**Value:**

**Connection:**

**CTA:**

## Sunday — Reflect + Preview

**Hook:**
```
Frame 1: "Weekly reflect + next week preview."
Frame 2: "What worked + what didn't + what's coming."
```

**Value:**
```
Frame 3: "Worked this week: [SPECIFIC THING]"
Frame 4: "Didn't work this week: [HONEST ITEM]"
Frame 5: "Reader theme that kept coming up: [COMMON TOPIC FROM DMs]"
Frame 6: "Next week's frame: [PREVIEW]"
Frame 7: "Next week's content: [1-2 specific teases]"
```

**Connection:**
```
Poll: "What do you want me to focus on next week?" — Option A / Option B
```

**CTA:**
```
"See you Monday. Have a real Sunday first."
```

---

## Slot-Fill Rules

- [COMPRESSED INSIGHT] should pass the tattoo test (readable in 3 seconds)
- Personal stories must be real
- Case notes must be dated and specific
- Reader questions must be actual (paraphrased OK; fabricated = violation)

---

## Visual Rhythm

- Monday / Wednesday / Friday: higher-design frames (carousel-like)
- Tuesday / Thursday: mixed
- Saturday: lo-fi personal style
- Sunday: quiet, muted palette

---

## Cadence Warning

Don't run daily value for 4 consecutive weeks without variation. Rotate with:
- Week N: daily value
- Week N+1: case study heavy
- Week N+2: lead magnet
- Week N+3: community building

---

## Common Failure Modes

- Insight isn't compressed (too wordy for the tattoo test)
- Personal Saturday drifts into pitch
- No-CTA rule violated on Saturday
- Sunday reflects without actually reflecting (generic wrap)
- Daily themes break character (loses rhythm)
