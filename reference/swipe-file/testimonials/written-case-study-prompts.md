# Written Case Study Prompts

> **8 prompts for collecting written testimonials / case-study contributions.** Async-friendly. Produces editable material across channels.

---

## When To Use Written Over Video

- Client is private and won't do video
- Client prefers async
- You need material you can easily excerpt for email / landing pages
- Time-zone mismatch makes video hard

Written testimonials have a trade-off: less trust signal than video, but higher edit-ability and cleaner output.

---

## The Prompts

Send as a shared document the client can fill in over 2-3 days.

### Prompt 1 — The Starting State
```
In 150-250 words, describe where your business was when we first started working together. 

What was working. What wasn't. What was the specific frustration that made you engage.

Be specific — numbers, situations, the texture of the week.
```

### Prompt 2 — The Decision
```
In 100-200 words: what made you decide to engage vs keep searching / waiting?

Include any skepticism you had. The objections that almost stopped you.
```

### Prompt 3 — The Process
```
In 200-400 words: describe what the engagement was like.

First 30 days. First 60 days. The specific work. What you did alone. What we did together.

Include moments of friction if there were any.
```

### Prompt 4 — The Specific Outcomes
```
Please share as much of the following as you're comfortable publishing:

— Starting revenue / key metric
— Ending revenue / key metric
— Timeframe
— Hours worked (before / after, if meaningful)
— Any other metric that moved

Anonymize if needed; I'll follow your lead on specificity.
```

### Prompt 5 — The Unexpected
```
In 100-200 words: what changed that you weren't expecting? 

Something beyond the metrics — operational, emotional, strategic.
```

### Prompt 6 — The Honest Caveat
```
In 100-200 words: what's been hard about this? 

Where have you hit friction? What hasn't worked as well as you'd hoped?

This matters — case studies without caveats read as marketing, not as testimony.
```

### Prompt 7 — The Recommendation Filter
```
In 100-200 words:

Who would you recommend this to specifically? (Describe the ICP as you'd describe it.)

Who wouldn't you recommend it to? What kind of operator or situation would make this a bad fit?
```

### Prompt 8 — The Raw Sentence
```
Finish this sentence:

"If a friend asked me at dinner what working with [YOUR NAME / BRAND] was actually like, I'd tell them: ______________."

This is the one line most likely to end up as a banner quote. Make it honest.
```

---

## Editing Protocol

Once the client submits:

1. Read the whole thing without editing
2. Confirm with the client: "Here's what I'm planning to use where."
3. Extract:
   - 1 full case study (800-1500 words) — sales page / long-form
   - 1 condensed version (200-400 words) — landing page
   - 3-5 quote pulls — social / email / ads
   - 1 single-sentence banner — homepage / sales page hero
4. Send back to client for approval on excerpts
5. Get written sign-off on specific uses

---

## Editing Ethics

Allowed:
- Light copy editing for clarity (fixing typos, smoothing grammar)
- Cutting for length
- Reordering for narrative flow

Not allowed:
- Changing meaning
- Adding content the client didn't write
- Combining quotes in ways that misrepresent
- Omitting qualifiers or caveats

When in doubt, ask the client.

---

## Turnaround Realism

Don't expect 24-hour turnaround on a written case study. Realistic timing:
- Initial request to draft: 1-2 weeks
- Draft to final: 1-2 revisions, 1-2 weeks
- Total end-to-end: 2-4 weeks

Don't pressure. Quality testimonials come from clients who had time to think.

## Permissions Document

Include with the prompts:

```
By submitting this document, I authorize [YOUR NAME / BRAND] to:

[ ] Use excerpts on website / sales page
[ ] Quote on social media (linked in post)
[ ] Include in email marketing
[ ] Use in paid advertising
[ ] Include in pitch decks / private materials only

What may be identified:
[ ] My name
[ ] My business name
[ ] Specific numbers I've shared
[ ] My industry / role

I retain the right to revoke permission with [N]-day written notice, 
after which [YOUR NAME] will cease new use (existing use may remain).

Signature + date
```

---

## Compensation Disclosure

If compensated in any way, disclose per FTC. Example:

> "This case study reflects results for [NAME], who received [X] in exchange for this contribution."

---

## Common Failure Modes

- Over-leading the prompts
- Skipping the caveat prompt
- Editing that changes meaning
- Missing permission document
- Publishing excerpts before client approval
- Pressuring turnaround
