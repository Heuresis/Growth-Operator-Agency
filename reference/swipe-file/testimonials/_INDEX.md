## When To Use Which

### Use Video Prompts when
- Client comfortable on camera
- You need highest-trust signal
- Sales page / VSL social proof section

### Use Written Prompts when
- Client prefers async
- You need flexibility for copy / email use
- Testimonial will be edited for different channels

### Use DM Screenshot anatomy when
- You have genuinely strong DMs
- Reader is skim-scanning
- Social proof on landing page or IG story

### Use Email Screenshot anatomy when
- Email has more depth than a DM
- Client opted in to share
- You need "she actually wrote this unprompted" signal

### Use Revenue Dashboard when
- Client has clean revenue / metric screenshots
- You have permission
- Numbers are the whole point

### Use Skeptic-Convert when
- Prospect audience is skeptical
- Client entered with objections
- Objection-handling is the asset

### Use Specific Metric when
- One number carries the whole testimonial
- Short-form platforms
- Proof-dominant use case

### Use Life Change when
- Client's life (not just business) changed
- Long-form platforms
- Trust/authority build

### Use Identity Shift when
- Client became someone new (operator vs owner, etc.)
- Identity-based ICP
- Community/cult methodology

### Use Peer Recommendation when
- Client is respected in your ICP's space
- Referral-style framing
- "People like you choose this"

## Testimonial Integrity

Per `INVARIANTS.md`:
- Never fabricate testimonials
- Always get written permission before publishing
- Never edit in a way that changes meaning
- Anonymize if requested (use role/archetype, not name)
- Include context (timeframe, what they did, what they got)

FTC compliance:
- Disclose compensation if any
- Note results aren't typical if they aren't
- Include timeframe + method

---

## Collection Cadence

Ask for testimonials at 3 moments:
1. **Early win** (first noticeable result) — short, specific, pattern-setting
2. **Midpoint** (halfway through engagement or halfway to goal) — process-focused
3. **Completion** (end of engagement or milestone hit) — full arc

Three testimonials per client = 3 different angles for different channels.
