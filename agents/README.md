41 agent role definitions organized as a real company: 1 CEO (`growth-ceo`), 7 department heads (foundations, marketing, nurture, sales, launch, scale, partnerships), and 33 specialists (ICP builder, offer architect, VSL writer, closer, etc.).

Each agent file declares its role, scope, decision authority, escalation paths, and the skills it owns. The roster is the org chart in code.
