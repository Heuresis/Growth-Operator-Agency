# skills/build-funnel/variants/

Per-archetype playbooks for the 15 funnel types `/build-funnel` can produce. Each file is operator-grade: the archetype's best use, the 5-stage structure (awareness / interest / consideration / decision / action), traffic sources, creative requirements, CVR benchmarks, tech stack, common failure modes, and economics.

## How to choose

Run `/build-funnel` and the skill gates on offer + audience compartments + traffic source + cycle length. If unsure, default:

- **Cold traffic + high-ticket** → VSL funnel or application funnel
- **Warm list + launch window** → challenge funnel or multi-video pre-launch
- **Ongoing evergreen** → webinar funnel or tripwire funnel
- **Founder brand, small team** → DM-to-close or newsletter-to-offer
- **Premium / exclusive positioning** → deposit-to-call or application funnel
