Per-type playbooks for the 7 X thread formats `/write-x-thread` can produce.

| File | Thread type | Best for |
|---|---|---|
| [educational-thread.md](educational-thread.md) | Educational / teaching | Authority build, bookmark-bait |
| [listicle-thread.md](listicle-thread.md) | Listicle / numbered | Shareability, scan-friendly |
| [case-study-thread.md](case-study-thread.md) | Case study breakdown | Proof, credibility, specific |
| [contrarian-thread.md](contrarian-thread.md) | Contrarian / reframe | Engagement, quote-tweet, reach |

| [personal-narrative-thread.md](personal-narrative-thread.md) | Personal story | Connection, emotional arc, trust |
| [curation-thread.md](curation-thread.md) | Curation / roundup | Traffic to others, reciprocity, low-effort |

## Structural baselines (across all types)

- **Tweet 1:** ≤ 280 chars, hook that promises the deliverable. 70% of reach depends here.
- **Tweets 2–N-1:** one idea per tweet, each quotable standalone. Specific beats generic.
- **Tweet N:** close + CTA (bookmark / follow / reply). No link in the final tweet if organic reach matters.
- **Ideal thread length:** 5–9 tweets. Shorter threads under-perform on reach; longer threads lose readers mid-way.
