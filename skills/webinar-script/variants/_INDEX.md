# skills/webinar-script/variants/

Per-format playbooks for the 6 webinar / challenge formats `/webinar-script` can produce.

| File | Format | Duration | Best for |
|---|---|---|---|
| [live-webinar.md](live-webinar.md) | Live webinar | 60–90 min | Real-time Q&A + launch momentum |
| [evergreen-webinar.md](evergreen-webinar.md) | Evergreen automation | 45–75 min | Perpetual funnel, paid-ad scaling |
| [three-day-challenge.md](three-day-challenge.md) | 3-day challenge | 60–90 min × 3 | Fastest belief arc + shortest runway |
| [five-day-challenge.md](five-day-challenge.md) | 5-day challenge | 60–90 min × 5 | Default challenge duration |
| [seven-day-challenge.md](seven-day-challenge.md) | 7-day challenge | 60–90 min × 7 | Deepest transformation; highest no-show risk |
| [workshop.md](workshop.md) | Premium workshop | 90–180 min | Paid workshop ($47–$997) with mid-ticket upsell |

## How to choose

- **60–90 min format** → live or evergreen webinar
- **Multi-day belief arc** → 3/5/7-day challenge (longer = deeper; shorter = less no-show risk)
- **Paid attendance filter** → workshop (higher quality attendee, higher close rate per)
