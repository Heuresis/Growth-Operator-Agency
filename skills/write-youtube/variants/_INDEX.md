# /write-youtube — Variant Index (7 Video Types)

> **Purpose.** Per-variant playbooks for each of the 7 canonical YouTube long-form video types. Each file contains: title formula, thumbnail SOP, script structure with timestamps, CTA placement, retention curve analysis, chapter markers, end-screen strategy, loop management, production notes, and at least one worked example.

## Decision Tree

```
IF conversion goal is paid offer AND distribution is organic:
    → VSSL / Hidden-Pitch (4)
ELIF conversion goal is pre-launch trust AND 8+ case studies exist:
    → Proof Showcase (6)
ELIF conversion goal is search-evergreen AND topic is task-based:
    → How-To Tutorial (3)
ELIF conversion goal is search-evergreen AND topic is conceptual:
    → Educational Deep-Dive (1)
ELIF conversion goal is decision-stage conversion AND 1 strong case study exists:
    → Case Study Breakdown (2)
ELIF conversion goal is identity association AND existing audience:
    → Lifestyle / Authority (5)
ELIF conversion goal is broad reach AND creator can sustain a serial arc:
    → Challenge Progression (7)
ELSE:
    → Default Educational Deep-Dive (1) — broadest applicability for most creator stages
```

## Cross-Variant Invariants

All 7 variants enforce:

- Hook Specificity ≥ 8 in the first 15 seconds.
- CTR 4%+ target, AVD 40-55%+ target (variant-specific).
- Title + Thumbnail complementary — two distinct information messages.
- 3+ `phrases_to_use` from Compartment 1 per 5-minute segment.
- Zero items from `spec/BANNED-VOCABULARY.md`.
- 3-5 curiosity loops opened and closed before outro (except VSSL and Lifestyle, which use fewer).
- Binge Loop outro for all except VSSL (which funnels to offer).
- Chapter markers in description with named, searchable chapter titles.
- Voice-match verified via read-aloud pass.
- Compliance pass — Truth Gate, FTC disclosures, no income guarantees.

## Failure Modes Common Across Variants

1. Title + thumbnail redundant — they convey the same message, wasted information.
2. Hook doesn't pay off the title promise in the first 60 seconds — CTR fine, retention dies.
3. Body drifts from the stated roadmap — viewers abandon at the pivot.
4. Loops opened but not closed — feels forgotten, not deliberate.
5. CTA stacking — 3 asks in the outro. Viewers do none.
6. Length padding — video stretched beyond the content. Retention drops proportionally.
7. Thumbnail clickbait that the video doesn't deliver — subscribe rate tanks.
8. Voice drift — creator voice replaced by "YouTube voice."
9. Banned vocabulary leakage in 3,000-6,000-word script is common — requires regex sweep pre-publish.
10. Missing chapter markers — reduces watch-time (viewers scrub to uninteresting parts and leave).

## End-Screen Strategy Summary

- Educational Deep-Dive: next video (within-YouTube only, no funnel).
- Case Study Breakdown: next case study in series.
- How-To Tutorial: next tutorial in workflow (template link in description).
- VSSL: offer funnel link + next video (for option-1 viewers).
- Lifestyle / Authority: next lifestyle / related personal video.
- Proof Showcase: full teardown of most dramatic case.
- Challenge Progression: next episode in the series.

## Hand-off to Editor (Universal)

Every YouTube video shipped to editing includes:

- Full script (1,400-6,750 words depending on variant).
- Title + A/B variants.
- Thumbnail concept + variant.
- Chapter markers with named, searchable chapter titles.
- Retention defenses mapped to chapter turns.
- Loop map with open / close timestamps.
- B-roll / screenshot / graphics list with timestamps.
- End-screen layout.
- Description with SEO-forward copy + FTC disclosures.
- Compliance check signed off.
